# VSCode Shortcuts

`alt + z`: 自动换行

`ctrl + d`： 多选, 按 `esc` 退出多选

`shift + alt + down`: 复制粘贴一整行

`ctrl + shift + p + 输入 fold all`: 折叠所有代码块

`ctrl + g + 输入数字`: 跳转到 \[数字\] 行

`ctrl + k + o`: 打开文件夹

`ctrl + l` 全选一行

# VSCode Terminal

# snippets

左下角齿轮 - snippets

输入 vue -> 自动跳转 vue.json

```json
{
    "vue3": {
        "prefix": "vue3",
        "body": [
            "<template>",
            "",
            "</template>",
            "",
            "<script setup lang=\"ts\">",
            "",
            "</script>",
            "",
            "<style scoped>",
            "",
            "</style>",
            ""
        ],
        "description": "vue3 template"
    }
}
```

以后直接输入 `vue3` 就会自动创建模板
