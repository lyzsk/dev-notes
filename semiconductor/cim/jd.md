# job description

职位描述 / 岗位职责

详细说明该职位的工作内容、职责范围、任职资格（如学历、技能、经验要求）以及薪资福利等信息的文档

## EAP 组

EAP 系统工程师

岗位职责:

负责公司 EAP 相关系统 (EAP, APC, FDC, RCM) 开发、管理和维护等

1. 负责系统各模块的设计、开发、配置、测试、安装、维护和升级; 监控系统的运行状况, 持续优化系统功能, 发现并解决系统故障等，确保系统安全性
2. 负责 收集和分析用户需求, 结合系统现有状况提供方案
3. 负责系统功能、算法开发及优化, 完成功能测试及上线验证
4. 负责确定系统功能验收规格, 核对系统开发功能符合用户需求, 落实设备工程系统的品质和对接工程确认版本上线节点

岗位要求:

1. 计算机、软件工程相关专业, 全日制本科以上学历, 有半导体工作经验优先
2. 掌握 C#/C/C++/VB 等至少一门开发语言; 掌握 Oracle/MySQL 数据库基础及 SQL 基本使用规范; 掌握 SECS/GEM 协议知识及协议应用规范
3. 具备较强的业务理解能力、解决问题的能力, 和团队协作能力和自我学习能力
4. 有开发经验、系统架构设计经验者优先

## MES 组

MES 系统工程师

负责公司 MES 相关系统 (MES, RMS, SPC, AMS, PMS) 开发、管理和维护等

岗位职责:

1. 负责系统代码设计开发、测试、部署与维护
2. 负责系统业务需求调研、分析、业务流程图设计、功能原型设计
3. 负责系统与其他 CIM 子系统的对接设计及数据维护
4. 负责系统用户培训, 上线后日常运维等

岗位要求:

1. 计算机、软件工程相关专业, 全日制本科以上学历, 有半导体工作经验优先
2. 精通使用 C#/Java 等编程语言, 能自行开发辅助性工具
3. 精通使用 SQL 语句, 熟练掌握 Oracle/MySQL 等常见数据库
4. 精通主流 MQ 中间件，熟练掌握软件系统测试方法与工具
5. 熟悉自动化流程设计与开发, 熟悉 MES/EAP/FDC 等 CIM 系统常用功能
6. 具备良好的文档编写能力、沟通表达能力和团队合作精神
7. 有开发经验、系统架构设计经验者优先
