## YMS Function List

### 1. 系统管理

#### 1.1 脚本管理

支持脚本文件的全生命周期操作.

- **脚本文件操作**: 支持新建脚本、打开脚本、保存脚本、脚本另存为、关闭脚本.

#### 1.2 语言设置

支持系统界面的多语言切换.

- **多语言切换**: 支持中文、英文界面语言切换.

#### 1.3 系统帮助

提供完善的系统帮助与版本信息查询.

- **帮助信息查看**: 支持查看使用手册、更新日志及版本号.

#### 1.4 退出系统

支持安全退出系统并保护未保存的工作内容.

- **退出保护**: 支持退出系统, 退出时提示保存当前操作.

### 2. 工具管理

#### 2.1 用户信息维护

支持系统用户信息的统一维护管理.

- **用户信息管理**: 支持新增、编辑、删除用户信息, 支持变更用户归属的用户组.

#### 2.2 用户组维护

支持用户组的灵活维护管理.

- **用户组管理**: 支持新增、编辑、删除用户组.

#### 2.3 功能维护

支持系统功能项的同步与启停控制.

- **功能项管理**: 支持同步系统功能, 支持启用、关闭系统功能.

#### 2.4 功能包维护

支持功能包的组织与管理.

- **功能包管理**: 支持新增、编辑、删除功能包, 支持添加功能至所属功能包.

#### 2.5 公共脚本管理

支持公共路径脚本的访问权限管控.

- **脚本权限设置**: 支持设置公共路径脚本访问权限, 支持按用户、用户组赋权.

#### 2.6 权限管理

提供功能粒度的权限管控能力.

- **访问权限控制**: 支持设置用户或用户组对功能、功能包的访问权限.

#### 2.7 排程管理

支持分析脚本的自动化排程执行.

- **自动排程**: 支持设置排程任务, 自动运行脚本并输出报告.

#### 2.8 产品光罩布局管理

支持产品光罩 (Reticle) 布局的定义与管理.

- **光罩布局管理**: 支持设置产品光罩布局定义信息, 管理产品光罩布局.

#### 2.9 CP 数据产品定义管理

支持 CP 测试数据产品定义的统一维护.

- **CP产品定义**: 支持设置维护 CP 数据产品定义信息.

#### 2.10 WAT 数据 SITE 定义管理

支持 WAT 数据 Site 定义的统一维护.

- **WAT Site定义**: 支持设置维护 WAT 数据 SITE 定义信息.

#### 2.11 Zone 布局维护

支持 Wafer Zone 布局定义的维护管理.

- **Zone布局定义**: 支持设置维护 Zone 布局定义信息.

#### 2.12 DSA 定义维护

支持 DSA 定义的维护与容差配置.

- **DSA定义管理**: 支持设置维护 DSA 定义, 支持 By FAB、产品、Layer、EQP 设置 Tolerance.

#### 2.13 缺陷分类定义维护

支持缺陷分类体系的标准化定义.

- **缺陷分类定义**: 支持设置维护缺陷分类定义信息.

#### 2.14 Cluster 定义维护

支持缺陷 Cluster 判定规则的定义维护.

- **Cluster定义管理**: 支持设置维护 Cluster 定义信息, 包括 min count、Tolerance.

#### 2.15 Repeat 定义维护

支持重复缺陷 (Repeat) 判定规则的定义维护.

- **Repeat定义管理**: 支持设置维护 Repeat 定义信息, 包括 min count、Tolerance.

#### 2.16 缺陷分类映射定义维护

支持跨 FAB 缺陷分类的映射统一管理.

- **分类映射管理**: 支持 By FAB 设置不同的缺陷分类映射至同一缺陷分类.

### 3. 数据导入

#### 3.1 文件导入

支持本地文件的便捷数据导入.

- **文件批量导入**: 支持通过 Excel、CSV 格式文件导入数据, 支持多文件批量导入.

#### 3.2 DB 导入

支持多数据源、多条件组合的数据库数据导入.

- **条件组合筛选**: 支持设置不同的条件 / 时间范围组合筛选.
- **时间规则查询**: 支持日历时间查询, 支持自定义时间规则.
- **通配符查询**: 查询条件支持通配符.
- **多数据源接入**: 支持选择从 CP, WAT, FT, WIP, Inline 等不同的数据源查询数据.

#### 3.3 SQL 查询

支持自定义 SQL 语句的灵活数据导入.

- **SQL直连导入**: 支持通过编写 SQL 语句实现从 DB 中导入数据.

### 4. 数据处理

#### 4.1 列转行

支持数据结构的行列转换.

- **列转行**: 支持数据的行列转换, 如列转行.

#### 4.2 行转列

支持数据结构的行列转换.

- **行转列**: 支持数据的行列转换, 如行转列.

#### 4.3 数据连接

支持多数据集之间的合并与运算.

- **数据合并运算**: 支持不同类型数据的左右合并、上下合并、相减.

#### 4.4 补值

支持缺失数据的灵活补值处理.

- **缺失值补值**: 支持统计值补值与自定义补值.

#### 4.5 数据分组

支持多维度的数据分组处理.

- **灵活分组**: 支持数据的自定义分组、条件式分组.

#### 4.6 数据筛选

支持规则化与统计化的数据过滤.

- **数据过滤**: 支持自定义规则过滤数据, 支持使用统计方法筛选数据.

#### 4.7 透视表

支持数据的聚合透视分析.

- **数据透视**: 支持行列聚合、数据透视功能.

#### 4.8 数据导出

支持处理结果的多格式导出.

- **多格式导出**: 支持数据导出为 Excel, CSV 格式.

### 5. MAP 分析

#### 5.1 CP/Sort/WAT Map

提供完整的晶圆图谱 (Wafer Map) 分析能力.

- **Map类型支持**: 支持查看良率 Map、参数 Map、WAT Map.
- **多层级分析**: 支持 Lot、Wafer 级别的分析, 支持单片以及多片叠图的分析.
- **Zone分析**: 支持 Wafer Zone 分析, 支持用户自定义分区计算 Yield、Parameter.
- **统计分析**: 支持 Bin 的统计分析与 Parameter 数据的统计分析.
- **分布图表**: 支持 Bin 分布图表功能 (如柱状图), 支持 Parameter 图表分析 (如直方图).
- **颜色梯度**: Sort Map 颜色梯度 (热力图) 支持 By Lot、Wafer 划分.
- **多片合并**: 支持多 Wafer Map 合并之后显示 Bin、Parameter 的分布.
- **WAT Map绘制**: 支持 WAT Map (5 Site / 9 Site) 的绘制.
- **光罩叠加**: 支持 Bin Map / Sort Map / WAT Map 和光罩叠加分析.

#### 5.2 样式功能

提供丰富的 Map 展示与交互样式能力.

- **结果导出**: 支持 Wafer Map 导出到 Excel, PowerPoint, 支持复制 Wafer Map 到剪贴板.
- **交互提示**: Wafer Map 支持鼠标悬停数据内容的提示显示.
- **视图操作**: Wafer Map 支持缩放、旋转, 支持 Die / Site / Reticle Level 的数据视图.

#### 5.3 Ink

支持晶圆 Ink 标注与结果输出.

- **Ink操作**: 支持选择不同的规则对 Map 进行 Ink 操作并导出 Text Map.

### 6. 基础分析

#### 6.1 相关性分析

支持变量间的相关性量化分析.

- **Correlation分析**: 支持 Correlation (Pearson, Spearman) 分析功能.

#### 6.2 数据汇总

支持全面的描述性汇总统计.

- **汇总统计**: 支持数据汇总统计, 包括均值、标准差、最大值、最小值、求和、个数、四分位数、百分位、IQR、偏度、峰度.

#### 6.3 单因子变异数分析

支持单因素方差分析.

- **One Way ANOVA**: 支持单因素方差分析 One Way ANOVA 分析功能.

#### 6.4 单因子变异数分析 (重复测量)

支持重复测量场景的单因素方差分析.

- **重复测量ANOVA**: 支持单因素方差分析 One Way ANOVA (重复测量) 分析功能.

#### 6.5 双因子变异数分析

支持两因素方差分析.

- **Two Way ANOVA**: 支持两因素方差分析 Two Way ANOVA 分析功能.

#### 6.6 描述性统计量计算

支持描述性统计的图形化呈现.

- **描述性统计绘图**: 支持描述性类别统计, 包括直方图、盒须图绘制.

#### 6.7 正态性检验

支持数据分布的正态性验证.

- **Normality Test**: 支持 Normality Test 分析功能.

#### 6.8 共通性分析

支持多维度共通性 (Commonality) 分析.

- **共通性分析**: 支持共通性分析功能, 支持根据不同的字段排序优先级.

### 7. 多变量分析

#### 7.1 主成分分析

支持主成分降维分析.

- **PCA分析**: 支持 Principle Component Analysis (PCA) (with loadings, scores, scree plot, scatter plot) 分析.

#### 7.2 偏最小二乘回归

支持偏最小二乘回归建模分析.

- **PLS回归**: 支持 PLS Regression (with X/Y loadings, scores, and VIP) 模型分析.

#### 7.3 多元回归

支持多元线性回归建模分析.

- **多元线性回归**: 支持 Multiple Linear Regression 分析功能, 支持设置多项式变量.

#### 7.4 决策树

支持决策树模型分析.

- **Decision Tree**: 支持 Decision Tree 分析.

### 8. 假设检验

#### 8.1 T 检验

支持单样本均值假设检验.

- **One Sample T-Test**: 支持 One Sample T-Test 分析功能.

#### 8.2 F 检验

支持联合假设检验.

- **F Test**: 支持 F Test (联合假设检验) 分析功能.

#### 8.3 两样本集成检验 (PCRB)

支持工艺变更前后差异的定量评估.

- **PCRB检验**: 支持辅助用户对工艺变更前后的样本间差异进行定量分析.

### 9. 图形绘制

#### 9.1 趋势图

支持多形态的趋势图绘制.

- **趋势图绘制**: 支持趋势图, 包括 Line, Line+Point, Bar, BarStacked.

#### 9.2 时序图

支持时间序列数据的可视化.

- **时序图绘制**: 支持时序图.

#### 9.3 盒须图

支持分布特征的盒须图展示.

- **盒须图绘制**: 支持盒须图, 盒须图支持显示均值、标准差、四分位数等统计值.

#### 9.4 散点图

支持变量关系的散点图展示.

- **散点图绘制**: 支持散点图.

#### 9.5 矩阵散点图

支持多变量两两关系的矩阵化展示.

- **矩阵散点图绘制**: 支持矩阵散点图.

#### 9.6 3D 散点图

支持三维空间的散点数据展示.

- **3D散点图绘制**: 支持 3D 散点图.

#### 9.7 均匀累积分布图

支持累积分布特征的展示.

- **CDF绘制**: 支持均匀累积分布图 (CDF).

#### 9.8 正态概率分布图

支持正态性评估的概率图展示.

- **NPP绘制**: 支持正态概率分布图 (NPP).

#### 9.9 帕累托图

支持关键少数因素的帕累托分析展示.

- **帕累托图绘制**: 支持帕累托图.

#### 9.10 直方图

支持数据分布的直方图展示.

- **直方图绘制**: 支持直方图.

#### 9.11 饼图

支持占比关系的饼图展示.

- **饼图绘制**: 支持饼图.

#### 9.12 曲面图

支持三维曲面数据的展示.

- **曲面图绘制**: 支持曲面图.

#### 9.13 等高线图

支持平面等值线数据的展示.

- **等高线图绘制**: 支持等高线图.

#### 9.14 样式功能

提供图表分组、布局与外观的深度定制能力.

- **分组分图**: 支持分组设置, 同一图内以不同颜色区分绘制不同组别的数据;支持分图设置, 同类型图表多图显示在同一窗口.
- **交互操作**: 支持图表的缩放, 支持复制图表到剪贴板.
- **外观定制**: 支持设置图表标题、图例、轴距、步距、颜色.
- **结果导出**: 支持导出图表至 Excel, PowerPoint.
- **辅助线**: 支持在图表上添加规格线、控制线.

### 10. 工作流

#### 10.1 样式功能

支持可视化工作流的编排与执行.

- **模块管理**: 支持新增、编辑、删除功能模块.
- **布局编排**: 支持拖拽模块布局, 支持一键垂直布局、水平布局.
- **数据承接**: 支持两模块之间连线, 承接数据.
- **一键执行**: 支持流程一键执行.

#### 10.2 流程决策

支持工作流中的条件分支控制.

- **分支决策**: 支持流程决策功能, 在工作流中判断流程分支走向.

### 11. 报告

#### 11.1 报告设置

支持报告的自定义生成、自动导出与分发.

- **报告定制**: 支持自定义报告版式布局、报告导出设定.
- **多格式导出**: 支持把分析结果导出到 Excel, PowerPoint, PDF 文件中.
- **自动报告**: 支持自动报告导出功能, 系统可以自动运行指定的分析模板生成并导出报告.
- **报告分发**: 报告生成后, 支持发送邮件、导出至共享文件夹机制.
