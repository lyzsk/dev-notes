# 数据采集
- >支持产品/设备关键参数/厂务低频率定时汇总/原材料/出货数据类型
- >支持多种采集方式，手动录入、EAP上传、文件解析、FTP、邮件解析
# 基础设置
- >支持整数、小数等数值类型，并可设置系统显示时保留数值的小数位数和Chart Y轴小数位数
- >支持K/NK/NC属性用于区分关键、非关键、不管控等Chart类型
- >支持Channel分组管理，可设定分组、子组、Report Group
- >支持参数版本管控
- >支持多种控制限，内规、客规、控制限、Target、Center Line、警告限、屏控限等
- >支持按照图表设置判异准则和Action（Email/Hold Lot/Down EQP/OCAP等），支持优先级触发OCAP，Action支持拓客制化扩展
- >按参数、Channel（数据采集维度）、CKC维度设定目标CPK
- >支持复制其他参数的Channel、CKC信息、规则、控制限
- >支持Excel模板导入/导出建模数据
- >参数、Channel（数据采集维度）、Chart三层建模数据的基本属性继承
# 管控维度
- >按工艺流程、设备、Recipe等多种主要信息设定数据采集维度（Channel），数据采集维度支持扩展
- >可动态启用或禁用特定Channel
- >支持【*】、【XX*】模式，数据采集时自动生成CKC
- >Channel维度可设置过滤条件，排除符合过滤条件的数据
# 计算型参数
- >支持计算型参数设定，一个参数可配置多个计算公式
- >支持跨站点参数计算
- >支持多种计算函数，以及加减乘除和常数
- >支持因子Raw Data计算，按照Index取部分原始值进行计算
# 判异规则
- >支持SPC八大判异准则
- >支持判异准则判定点数设定
- >支持上下限判异，OOS/OOC/OOR/OOT/OOW
- >支持连续N点异常等客制化规则
- >支持组合基本规则的客制化规则
# 收藏夹管理
- >支持按用户创建收藏夹，将用户关注的关键Chart存入收藏夹中，方便查找和查看。
- >支持按收藏夹批量打开Chart。
- >支持按收藏夹批量导出Chart。
# 图表样式管理
- >按参数、Channel、Chart等多个维度设置图表显示样式。
- >支持设置点的形状、大小、颜色等样式，包含正常点、异常点、备注点等。
- >支持设置线的类型、粗细、颜色等样式，包含USL、LSL、UCL、LCL等。
- >支持设置箱线图点位偏移量，让重叠数据左右偏移。
- >支持设置X轴Lable默认显示信息，包含时间、UNIT_ID、LOT_ID等。
- >支持设置X轴默认Lable显示数量、倾斜角度等。
# 图表管理
- >支持多种图表类型（X、Sigma、Rang、Move Range、Move Sigma、Move Avg、XBAR、XBAR_S、XBAR_R）
- >支持正态分布图显示
- >支持计数型Chart，P、NP、C、U图
- >支持按Channel/CKC查看Chart
- >支持多条件过滤Chart，参数名、Chart名、样本ID，支持保存常用筛选条件
- >支持不同样式显示点，正常点、异常点、备注点样式区分
- >支持Y轴调整，根据管控限或自定义范围调整Y轴，支持科学计数法、对数显示
- >可显示为时间、UNIT_ID、LOT_ID、T7 Code，可组合显示
- >Chart控制线使用历史数据进行展示
- >支持显示数据点详细信息，数据采集的主数据与辅助信息
- >支持显示Count/Avg/Std/StdP/K/Pp/Ppk/Cp/Cpk/Oos Count/Oos Ratio/Ooc Count/Ooc Ratio/Abnormal_Count/Abnormal Ratio等指标
- >支持高亮A/B/C区
- >可对数据点添加备注，查看该点过往所有备注记录
- >支持数据点临时/永久隐藏，临时/永久取消隐藏
- >隐藏点或过滤数据点后按照过滤后数据重新计算Cpk、Avg等相关计算数据
- >支持一键导出Chart与原始数据，导出格式Excel/PPT
- >支持多时间维度排序，Process Time、Measure Time、System Time
- >支持按点数、时间周期过滤，按重量、离散点、屏控限等条件批量过滤Chart数据
- >支持按辅助信息、CKC分段显示控制图数据
- >基于当前Chart数据推荐控制限，计算公式Avg±3Sigma
- >支持选择数据点手动触发OCAP
- >支持从异常点跳转至OCAP处理页
# 自动计算
- >支持设置计算公式，数据过滤条件
- >支持自动/手动计算，自动计算周期月度/季度
- >计算结果以报表形式展示
