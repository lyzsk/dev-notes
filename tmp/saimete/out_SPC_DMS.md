## SPC Function List

### 1. 数据采集

支持多类型、多方式的生产与质量数据采集.

- **数据类型**: 支持产品、设备关键参数、厂务低频率定时汇总、原材料、出货等多种数据类型的采集.
- **采集方式**: 支持手动录入、EAP 上传、文件解析、FTP、邮件解析等多种采集方式.

### 2. 基础设置

支持 SPC 建模所需的数值类型、Chart 属性、控制限及建模数据的统一管理.

#### 2.1 参数与 Chart 属性

- **数值类型**: 支持整数、小数等数值类型, 可设置系统显示保留的小数位数及 Chart Y 轴小数位数.
- **Chart 分类属性**: 支持 K/NK/NC 属性, 用于区分关键、非关键、不管控等 Chart 类型.
- **Channel 分组管理**: 支持 Channel 分组管理, 可设定分组、子组、Report Group.
- **版本管控**: 支持参数版本管控.
- **目标 CPK 设定**: 支持按参数、Channel(数据采集维度)、CKC 维度设定目标 CPK.

#### 2.2 控制限与判异 Action

- **多种控制限**: 支持内规、客规、控制限、Target、Center Line、警告限、屏控限等多种控制限.
- **判异准则与 Action**: 支持按照图表设置判异准则和 Action(Email/Hold Lot/Down EQP/OCAP 等), 支持按优先级触发 OCAP, Action 支持客制化扩展.

#### 2.3 建模数据管理

- **批量建模**: 支持 Excel 模板导入/导出建模数据.
- **配置复制**: 支持复制其他参数的 Channel、CKC 信息、规则、控制限.
- **属性继承**: 支持参数、Channel(数据采集维度)、Chart 三层建模数据的基本属性继承.

### 3. 管控维度

支持按多种业务信息定义数据采集维度(Channel)并进行灵活管控.

- **多维度定义**: 支持按工艺流程、设备、Recipe 等多种主要信息设定数据采集维度(Channel), 且数据采集维度支持扩展.
- **动态启停**: 支持动态启用或禁用特定 Channel.
- **通配符模式**: 支持【*】、【XX*】模式, 数据采集时自动生成 CKC.
- **过滤条件**: Channel 维度可设置过滤条件, 排除符合过滤条件的数据.

### 4. 计算型参数

支持基于公式的计算型参数定义与跨站点计算.

- **多公式配置**: 支持计算型参数设定, 一个参数可配置多个计算公式.
- **跨站点计算**: 支持跨站点参数计算.
- **计算函数**: 支持多种计算函数, 以及加减乘除和常数运算.
- **因子取值**: 支持因子 Raw Data 计算, 可按照 Index 取部分原始值参与计算.

### 5. 判异规则

支持标准 SPC 判异准则及灵活的客制化规则配置.

- **标准判异准则**: 支持 SPC 八大判异准则.
- **判定点数设定**: 支持判异准则判定点数设定.
- **上下限判异**: 支持 OOS/OOC/OOR/OOT/OOW 等上下限判异.
- **客制化规则**: 支持连续 N 点异常等客制化规则, 以及组合基本规则的客制化规则.

### 6. 收藏夹管理

支持按用户管理常用 Chart, 提升查找与批量操作效率.

- **个人收藏夹**: 支持按用户创建收藏夹, 将用户关注的关键 Chart 存入收藏夹, 方便查找和查看.
- **批量打开**: 支持按收藏夹批量打开 Chart.
- **批量导出**: 支持按收藏夹批量导出 Chart.

### 7. 图表样式管理

支持多维度、精细化的图表显示样式定制.

- **多维度样式设置**: 支持按参数、Channel、Chart 等多个维度设置图表显示样式.
- **数据点样式**: 支持设置点的形状、大小、颜色等样式, 包含正常点、异常点、备注点等.
- **控制线样式**: 支持设置线的类型、粗细、颜色等样式, 包含 USL、LSL、UCL、LCL 等.
- **箱线图偏移**: 支持设置箱线图点位偏移量, 让重叠数据左右偏移.
- **X 轴标签**: 支持设置 X 轴 Label 默认显示信息(时间、UNIT_ID、LOT_ID 等), 以及默认显示数量、倾斜角度等.

### 8. 图表管理

支持丰富的 Chart 类型、灵活的查询过滤及完整的数据点交互操作.

#### 8.1 图表类型与查询

- **计量型图表**: 支持 X、Sigma、Range、Move Range、Move Sigma、Move Avg、XBAR、XBAR_S、XBAR_R 等多种图表类型.
- **分布与计数型图表**: 支持正态分布图显示;支持 P、NP、C、U 等计数型 Chart.
- **Chart 查看与过滤**: 支持按 Channel/CKC 查看 Chart;支持按参数名、Chart 名、样本 ID 等多条件过滤 Chart, 并可保存常用筛选条件.

#### 8.2 图表显示

- **坐标轴调整**: 支持 Y 轴根据管控限或自定义范围调整, 支持科学计数法、对数显示.
- **X 轴内容**: X 轴可显示时间、UNIT_ID、LOT_ID、T7 Code, 可组合显示;支持 Process Time、Measure Time、System Time 等多时间维度排序.
- **历史控制线**: Chart 控制线支持使用历史数据进行展示.
- **数据点样式区分**: 支持正常点、异常点、备注点等不同样式显示.
- **统计指标显示**: 支持显示 Count/Avg/Std/StdP/K/Pp/Ppk/Cp/Cpk/Oos Count/Oos Ratio/Ooc Count/Ooc Ratio/Abnormal Count/Abnormal Ratio 等指标.
- **分区高亮**: 支持高亮 A/B/C 区.
- **分段显示**: 支持按辅助信息、CKC 分段显示控制图数据.
- **数据点详情**: 支持显示数据点详细信息, 包含数据采集的主数据与辅助信息.

#### 8.3 数据点操作

- **备注管理**: 支持对数据点添加备注, 并查看该点过往所有备注记录.
- **隐藏与恢复**: 支持数据点临时/永久隐藏, 以及临时/永久取消隐藏.
- **过滤重算**: 隐藏点或过滤数据点后, 按照过滤后数据重新计算 Cpk、Avg 等相关统计数据.
- **数据过滤**: 支持按点数、时间周期过滤, 以及按重量、离散点、屏控限等条件批量过滤 Chart 数据.

#### 8.4 控制限与 OCAP 联动

- **控制限推荐**: 支持基于当前 Chart 数据推荐控制限, 计算公式为 Avg±3Sigma.
- **OCAP 触发**: 支持选择数据点手动触发 OCAP, 并支持从异常点跳转至 OCAP 处理页.

#### 8.5 数据导出

- **一键导出**: 支持一键导出 Chart 与原始数据, 导出格式为 Excel/PPT.

### 9. 自动计算

支持周期性的统计指标自动计算与结果展示.

- **计算配置**: 支持设置计算公式及数据过滤条件.
- **计算方式**: 支持自动/手动计算, 自动计算周期支持月度/季度.
- **结果展示**: 计算结果以报表形式展示.

### 10. 报表

支持常用的质量统计报表.

- **Cpk 报表**: 提供 Cpk 统计报表.
- **报警率报表**: 提供报警率统计报表.
- **管控限报表**: 提供管控限统计报表.

## DMS Function List

### 1. 缺陷数据查询模块

支持多维度、多条件的缺陷数据检索与筛选.

- **多维度过滤**: 支持按时间过滤筛选缺陷数据, 支持产品、批次、晶圆、站点、配方等维度的数据过滤.
- **条件筛选**: 支持按数据内容及原始数据条件进行缺陷数据的筛选过滤.
- **结果过滤**: 支持设置仅呈现最终检测结果;支持筛选过滤有照片的检测记录.
- **Klarf 读取**: 支持直接读取 Klarf 档案进行查询分析.
- **显示模式**: 支持列表、行列定义等多种显示模式.
- **指标查询**: 支持查询缺陷分类结果以及 DDC/DDP 等指标.

### 2. 数据查询

支持原始数据查询与自定义图形化展示.

- **原始数据查询**: 支持查询原始数据.
- **自定义图形**: 用户可自行定义图形显示条件.
- **数据导出**: 查询结果支持汇出为 Excel、CSV.

### 3. 多图显示模块

支持多片 Wafer Map 的同屏显示与分析.

- **排布调整**: 支持自由调整显示的行列数量.
- **群聚分析**: 支持缺陷群聚(Cluster)分析.
- **颜色自定义**: 支持自由调整缺陷类别的显示颜色.
- **检测资讯**: 支持显示检测资讯.
- **抽样分析**: 支持抽样分析.

### 4. 缺陷晶圆图的单图与叠图显示模块

支持缺陷晶圆图(Wafer Map)的显示控制、交互操作、叠图分析与数据输出.

#### 4.1 Map 显示与交互

- **显示控制**: 支持显示检测资讯;支持调整缺陷晶圆图的显示细项条件, 缺陷显示条件可自由调整;支持放大或缩小缺陷晶圆图.
- **芯片显示**: 芯片资讯与芯片边界可选择是否显示.
- **框选操作**: 用户可在缺陷晶圆图中以任意形状选取缺陷.
- **测量工具**: 支持测量两个缺陷间的距离.
- **复制输出**: 支持一键复制 Map 至剪贴板.

#### 4.2 叠图与联动分析

- **区域叠图**: 支持晶圆中按区域或 Shot 等进行缺陷的叠图.
- **CP 叠图**: 支持 CP 测试结果与缺陷叠图.
- **区域统计**: 单图与叠图时支持区域划分与统计分析.
- **联动显示**: 缺陷晶圆图可与缺陷数据、统计图互动显示.
- **统计图调整**: 统计图支持改变各种显示条件, 如 X/Y 轴条件、开关缺陷是否有分类或缺陷尺寸等.
- **照片显示**: 缺陷照片支持多种显示模式(列表、行列定义、按条件排序等).

#### 4.3 数据输出与抽样

- **Klarf 输出**: 支持输出 Klarf 档案.
- **缺陷拍照**: 支持指定缺陷进行检查拍照.
- **自动抽样**: 支持自动缺陷抽样, 可配置多种多重条件.

### 5. 新增缺陷分析

支持前后测缺陷的对比呈现.

- **前后测对比**: 支持分别呈现前测、后测、前后测共同、后测新增等缺陷晶圆图.

### 6. 缺陷晶圆图的比较

支持缺陷差异的可视化比较.

- **差异着色**: 支持按后测增加、后测消失、前后测有差异等条件改变缺陷显示颜色.

### 7. 缺陷分类模块

支持缺陷的人工重新分类.

- **人工分类**: 界面上支持缺陷的重新人工分类.

### 8. 绘图模块

支持多种统计图表类型.

- **图表类型**: 支持各式趋势图、条形图、堆叠条形图、面积图、散点图、盒须图.

### 9. 缺陷照片浏览模块

支持缺陷照片的浏览、操作与重分类.

- **树状浏览**: 支持树状缺陷照片浏览.
- **照片操作**: 支持放大或缩小缺陷照片, 可点击照片进行重新分类.
- **照片堆栈**: 支持将同一颗缺陷的不同照片堆栈显示.

### 10. 动态缺陷根源分析模块

支持缺陷根源分析(DSA)的参数调整与结果汇总.

- **误差调整**: 用户可自行调整缺陷根源分析的缺陷位置误差范围.
- **汇总分析**: 提供缺陷根源分析的晶圆层级汇总与相关性汇总表.
- **命中对照**: 提供缺陷根源分析后的命中缺陷对照表.
