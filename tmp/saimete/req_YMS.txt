# 系统
- 多语言支持：繁体中文/简体中文/英文
- 系统提供高可用性设计并包含系统异常处理及故障恢复功能
- 系统在User 端运行，OS需支持Windows 7/8/10皆正常运作
- 可定时执行分析流程，并将执行后的报告储存在指定路径
- 可储存分析流程并再次执行
- 可弹性调整分析平台内各功能的分类与功能的权限控管
- 可支持Workflow模式
- 可设定寄送报表收件邮箱
- 支持分析过程中使用截屏购物车(Image Cart)，且能一键生成PPT报表
- 喜好设定 (接口样式/图型颜色/字体/大小)
- 提供流程Loop之运算功能
# 数据查询
- 可支持多种数据库如Oracle, MS SQL, PostgreSQL, DB2, Greenplum
- 查询功能提供透过 SQL 指令直接取得数据库内数据
- 查询接口开发工作, 需提供免编程工具，可根据管理者设定，使用拖拉方式轻易设定、调整后，自动生成重复使用之查询接口
- 使用者可自定义查询条件(包含产品、批号、制程站点、机台名称、日期时间等)
- 查询结果可被储存再利用，随时可调整且接续着下一步的分析
- 用户可以过滤或给予一定的规则查询部分的结果
- 查询结果可以输出
- 可使用通配符当作查询条件 (%, *)

# 图表
- 趋势图（趋势线、点图、条形图、堆叠条形图、百分比堆叠条形图、面积图）,时间序列图，饼图，柏拉图，箱形图，区间图，Delta图，散点图，3D散点图，散点图矩阵，直方图，正态概率图，P-P图，Q-Q图，曲面图，等高线图，色阶网格图，箱形趋势图，颜色映射散点图
- 趋势图、条形图、箱形图、箱形趋势图支持X轴合并显示（分组标签）
- 依自动分群条件一次绘制多张图表
- 支持从数据列生成规格线（含动态规格线功能）
- 高亮显示OOC与OOS数据点
- 可于多图模式根据OOC、OOS高光出违反的Chart
- 在图表上直接计算并显示Cpk值
- 根据设定条件绘制常数线，条件可参考其他表格或自行输入
- 趋势图支持自动侦测趋势转折点，并绘制分隔线(良率转折点、电测转折点)
- 散点图支持定义器件四点规格（FF/SS/SF/FS/中心点）

# 报表
- 提供接口设定各种报表逻辑，方便操作
- 报告输出格式: HTML, Excel (2003, 2007), CSV, Power Point(2003,2007)
- 自动报表定义方法支持两个表格合并数据显示于Excel报表
- 自动报表定义方法支持表格与图片(图表 /图片/Map )合并显示于Excel报表
- 能对多个表格与Chart, Map产出于Excel 报表中，产生Index Sheet, 并建立链接，方便浏览数据
- 支持自定义单页面报表布局(例：2*2, 3*2…)
- 支持在操作界面动态调整数据顺序、字段名称及字体大小
- PPT自动报表可支持多章节格式，且每章节可自定义不同格式
- PPT自动报表可于单页PPT内定义X轴与Y轴，将多张图片依照定义合并显示于一页PPT
- PPT自动报表可根据图片数量与排列逻辑自定义分页方式
- 可支持Email 寄送Report，并支持邮件内含HTML 报表内容
