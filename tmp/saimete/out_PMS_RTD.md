## PMS Function List

### 1. 机台保养 (PM Management)

#### 1.1 保养类型与触发机制

支持多种类型的维修保养模式及灵活的触发方式, 满足计划与非计划维保场景.

- **计划性保养**: 支持计划的维修保养管理.
- **非周期维修**: 支持非周期的维修管理及非计划维修保养任务.
- **时间周期保养**: 支持基于时间的周期性保养管理.
- **使用情况保养**: 支持基于使用情况的周期性保养管理, 与 EAP 整合, 接收对应参数值进行触发 (例如晶圆计数以及 RFTM).
- **混合类型保养**: 支持基于时间周期和基于使用情况相结合的混合类型保养管理.
- **API 创建维修单**: 支持通过 API 供其他系统创建临时的维修保养单.

#### 1.2 PM 模板与计划配置

支持标准化的保养模板统一管控与精细化的计划规则设定.

- **模板统一管控**: 计划性维护保养模板支持按照设备模板统一管控或者按照机台 ID 进行维护, 具备版本控制功能.
- **多层级配置**: 计划性维护支持按照设备、按照子设备进行配置.
- **Checklist 关联**: 支持选择所需的已建立设备检查清单模板, 并可一次添加多个设备检查清单模板.
- **触发参数设置**: 支持设置触发设备维保所需要的参数 (递增性以及替换性参数).
- **预警与逾期设置**: 支持设置设备维保的提前预警、规定时间以及逾期时间, 并可定义是否在逾期时自动设置机台维保超期状态.
- **命名规则**: 支持定义计划性维护保养所生成的设备维保任务命名规则.
- **浮动与静态规划**: 支持定义浮动型以及静态型 PM, 即根据 PM 完成时间或者固定的时间来重新规划下一次 PM 时间.
- **共享附件**: 支持添加 PM 过程中用户需要的共享附件 (例如 SOP 文件).
- **部品与关键配件**: 支持定义 PM 需使用的部品详细信息 (料号与描述信息), 以及定义 PM 计划时一定要更换的关键配件清单.
- **模板维护**: 支持增加、修改以及复制维修保养管理模板.
- **原因代码**: 支持设置 PM 保养管理所需要的原因代码.
- **预估时间与标题**: 支持设置 PM 保养所需要的预估保养时间以及标题.
- **签核逻辑**: 提供系统默认签核逻辑 (权限群组设定) 管理维修保养管理模板.

#### 1.3 PM Checklist 管理

支持设备检查清单模板的结构化配置与全生命周期维护.

- **模板配置**: 支持设备检查清单模板配置功能 (按设备模板或者按设备), 支持系统默认的签核逻辑.
- **步骤管理**: 支持添加、删除以及修改设备检查清单步骤.
- **导入导出**: 支持导出设备检查清单模板以及导入设备检查清单模板.
- **参数类型**: 支持设置必填以及选填参数.
- **参数提示**: 未填写必输入参数时, 完成会进行报警提醒.
- **规格卡控**: 支持规格限制以及检查 (是否需要 OOS 上下限卡控).
- **步骤附件**: 支持添加用户执行设备检查清单步骤时所需的共享附件 (如执行 PM 所需要的 SOP 文件或者图片).
- **复机检查**: 支持 PM 复机设备检查清单的检查执行步骤设置.
- **模板维护**: 支持增加、修改以及复制维修 PM 设备检查清单模板.
- **故障单关联**: 支持非计划性宕机 (UnScheduling Down) 手动建立故障单作业请求, 并手动添加所需设备检查清单.

#### 1.4 保养排程与预警通知

支持智能化的保养排程辅助与多级预警通知机制.

- **时间预测**: 支持预测维修保养的时间.
- **自动排程**: 周期性保养完成后自动安排下一个维修保养管理.
- **强制切换**: 具备达到强制维修保养条件逾期时, 强制切换设备/子设备可用性的功能.
- **PM 预约**: 支持预约 PM.
- **提前预警**: 通过用户定义的提前时间预先报警发送给预警系统.
- **到期报警**: 通过用户定义的设定时间报警发送给预警系统.
- **逾期报警**: 通过用户定义的逾期时间报警发送给预警系统.

#### 1.5 保养执行与关键配件卡控

支持完整的保养任务操作流转与关键配件更换防错卡控.

- **任务操作**: 维修保养任务支持开始、中止、取消、继续、完成、延迟等操作.
- **合并执行**: 关联的保养计划可以合并执行, 如周保养和月保养遇到时可以选择合并.
- **关键配件卡控**: 完成 PM 时检查需更换的关键配件是否已完成更换, 否则 PM 不可以结束; 更换可设置是否强制, 且需要填写备注.

#### 1.6 查询与报表

支持多维度的保养信息查询与清单导出.

- **组合查询**: 可按照设备、拟定的设备群、默认时间范围来查询.
- **计划与状态查询**: 支持查询维修保养管理计划及维修保养管理状态.
- **清单查看与导出**: 支持查看维修保养清单列表及导出维修保养清单列表.

#### 1.7 权限管理

支持配置与运行并重的权限管理机制及与 MES 的权限整合.

- **权限机制**: 提供用户权限的管理机制, 包括配置以及运行的权限管理.
- **MES 整合**: 权限管理模块与 MES 进行整合, 减少用户重复定义.

## RTD Function List

### 1. 用户权限管理

#### 1.1 用户管理

支持角色的全生命周期管理.

- **角色新建与查看**: 支持新建角色及查看角色.
- **角色复制**: 支持复制角色, 快速创建相似角色.
- **角色维护**: 支持编辑角色及删除角色.

#### 1.2 权限管理

支持用户账号的全生命周期管理.

- **用户新建与查询**: 支持新建用户及查询用户.
- **用户维护**: 支持编辑用户及删除用户.
- **密码管理**: 支持重置用户密码.

### 2. Report 管理

#### 2.1 分组管理

支持 Report 分组的组织与维护.

- **分组维护**: 支持添加分组、修改分组名称及删除分组.

#### 2.2 Report 编辑

支持 Report 的可视化开发、发布与权限控制.

- **可视化编辑**: 提供 Report 可视化编辑器.
- **版本与导出**: 支持 Report 保存与升版, 以及 Report 导出与导入.
- **DV 发布管理**: 支持上线/重建 DV, 以及下线/删除 DV.
- **权限与分组**: 支持 Report 权限设置及 Report 切换分组.
- **Report 查询**: 支持 Report 查询.

### 3. Macro 管理

#### 3.1 Macro 编辑

支持 Macro 的可视化开发与版本管理.

- **可视化编辑**: 提供 Macro 可视化编辑器.
- **版本管理**: 支持 Macro 保存与升版.
- **导入导出**: 支持 Macro 导出及 Macro 导入.

#### 3.2 Macro 发版

支持 Macro 的上线与下线发布管理.

- **上线发布**: 支持 Macro 上线 DV.
- **下线管理**: 支持 Macro 下线 DV.

### 4. 机台映射

#### 4.1 机台映射

支持机台映射关系的批量维护与编辑.

- **批量导入导出**: 支持机台映射的批量导入与导出.
- **映射编辑**: 支持机台映射编辑.

### 5. 全局变量

#### 5.1 全局变量

支持全局变量的批量维护与编辑.

- **批量导入导出**: 支持全局变量的批量导入与导出.
- **变量编辑**: 支持全局变量编辑.

### 6. 依赖关系查询

支持系统内对象依赖关系的查询, 便于变更影响分析.

- **依赖查询**: 支持依赖关系查询.

### 7. 历史记录

#### 7.1 历史记录

支持用户操作记录的追溯查询.

- **操作记录查询**: 支持查询用户操作记录.

### 8. DV 管理

#### 8.1 DV 管理

支持 DV 及其参数表的管理与异常处理.

- **DV 查询与删除**: 支持 DV 查询及删除.
- **参数表管理**: 支持参数表查询、删除及清空.
- **异常管理**: 支持 DV 异常管理.
