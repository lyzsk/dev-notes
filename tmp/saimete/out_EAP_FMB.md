## EAP Function List

### 1. 标准功能

#### 1.1 EAP模板管理

支持按设备 Buffer 类型、用途、工艺、通讯方式建立模板库, 沉淀标准化 Scenario, 提升后期开发与导入效率.

- **内置模板**: 支持 FixedBuffer, Inline, Furnace, Wet, InternalBuffer, Sorter, CarrierExchange, Bonding, FoupClean, FoupInspection, ReticleStocker, ReticleInspection, N2Purge, 开盒器等设备模板.
- **模板复用**: 503 台设备中 390 台 NewType 设备优先按模板沉淀复用.

### 2. 设备接入

#### 2.1 SECS Driver / 设备通讯

支持标准与非标通讯协议的设备接入, 保障通讯稳定可靠.

- **标准协议**: 支持 SECS-I, HSMS, SECS-II, GEM 等半导体标准通讯.
- **接入方式**: 支持 Serial, TCP, TCP_SECS1, UDP, PIPE, PLC, Barcode, RFID, SMIF 等接入方式.
- **通讯可靠性**: 支持大报文处理, 断线重连, 通讯超时与异常恢复.

### 3. GEM能力

#### 3.1 GEM基础状态机

支持完整实现 GEM 标准状态机及常用 Stream, 保证与设备状态同步.

- **状态机**: 实现 Communication, Control, Processing 三类状态机.
- **状态同步**: 支持 COMMUNICATING, OFF-LINE/LOCAL/REMOTE, IDLE/READY/EXECUTING 等状态同步.
- **常用 Stream**: 支持 S1/S2/S5/S6/S7/S10 等常用 Stream.

#### 3.2 事件、报警、变量与远程命令

支持 GEM 标准的事件、报警、变量采集与远程命令处理能力.

- **事件与变量**: 支持 Collection Event, Report, SV/DV/EC 数据采集.
- **报警管理**: 支持 Alarm Set/Clear 成对上报.
- **远程命令**: 支持 Remote Command START, STOP, PAUSE, RESUME, ABORT 等命令及 ACK 处理.

### 4. 流程控制

#### 4.1 Scenario流程配置

支持 EAP Scenario 的图形化配置与灵活的流程控制能力.

- **图形化配置**: 支持 EAP Scenario 的图形化配置, 显示和修改.
- **流程配置**: 支持异常流程, 动态事件配置, Port/Chamber/Recipe 路径配置, Inline 多设备配置.
- **作业控制**: 支持 FOUP/Cassette 优先级控制和 Wafer 级流程控制.

### 5. 业务校验

#### 5.1 帐料校验与作业管控

支持与 MES 交互完成帐料校验与作业管控, 确保生产准确性.

- **信息查询**: 支持向 MES 查询 Carrier, Lot, Equipment, Batch, EDC 等信息.
- **匹配校验**: 支持 Lot 与设备匹配校验, Recipe 校验.
- **作业动作**: 支持 JobIn, JobOut, CancelJobIn, HoldLot, BatchJob 等作业动作.
- **异常处置**: 校验不匹配时支持退 Port, Hold Lot, 报警等处理.

### 6. 载具管理

#### 6.1 LoadPort / Carrier控制

支持 Load Port 与 Carrier 的全流程控制, 覆盖 E84/E87 标准场景.

- **Load Port 控制**: 支持 300mm LoadPort, FOUP/FOSB/OpenCassette 的自动/手动模式控制.
- **E84/E87 流程**: 支持 Carrier 到达, ID 验证, SlotMap 验证, Proceed, Complete, Release, CarrierOut 等流程.

### 7. 配方协同

#### 7.1 Recipe / RMS集成

支持与 RMS 集成的 Recipe 全生命周期协同管理.

- **Recipe 解析**: 支持 RecipeBody, RecipeParameter 解析与上传.
- **RMS 协同**: 支持 RMS 请求抓取, 下载, 比对, 版本校验.
- **Run 货前校验**: 支持 Run 货前校验 RecipeID 存在性及与 MES/RMS 配方一致性.
- **信息展示**: 支持当前 Recipe 信息展示.

### 8. 数据采集

#### 8.1 EDC/FDC/AMS数据上报

支持多方式设备数据采集, 格式转换及向上位系统上报.

- **采集方式**: 支持 SECS/GEM, TCP/IP, File, Database, FTP, PLC 等方式采集数据.
- **采集内容**: 支持设备事件, 报警, 状态, 常量, 工艺参数等数据采集.
- **数据配置**: 支持数据格式转换与采集开关配置.
- **数据上报**: 支持向 FDC, AMS, MES 等系统上报.

### 9. 状态同步

#### 9.1 设备状态与控制同步

支持设备各类状态与上位系统实时同步, 并支持远程控制.

- **状态同步**: 支持设备通讯状态, Online/Offline, Local/Remote, Port Auto/Manual, 设备运行状态, Chamber 状态同步.
- **远程控制**: 支持 MES 侧状态查询和切换控制.

### 10. 物料追踪

#### 10.1 Carrier/Wafer/Buffer追踪

支持 Carrier, Wafer, Buffer 级别的物料流转追踪与信息上报.

- **过账上报**: 支持 Track-in, Track-out, Hold 等过账信息实时上报.
- **Buffer 追踪**: 支持 Buffer 管控及 Cassette 进出 Buffer 信息上报.
- **Wafer 追踪**: 支持 Wafer 进出 Chamber 信息采集及 Wafer History 采集.

### 11. 光罩管理

#### 11.1 Reticle管理

支持面向 12 寸光刻相关设备的光罩 (Reticle) 精细化管理.

- **光罩操作**: 支持光罩 Load/Unload 及 Reticle Library 状态展示.
- **使用追踪**: 支持光罩使用次数采集.
- **检测上报**: 支持 IRIS 检测数据上报.
- **状态验证**: 支持光罩预约及 Run 货前光罩状态验证.

### 12. 系统集成

#### 12.1 上位系统接口

支持与各类上位系统集成, 接口方式灵活多样.

- **集成范围**: 支持与 MES, RMS, FDC, APC, AMS, PMS, AMHS, RCM 等系统集成.
- **接口方式**: 支持 Message Bus, REST/Web API, TCP/IP, DB/File 等接口方式, 具体以客户 CIM 接口规范为准.

### 13. 客户端

#### 13.1 OUI / EAP客户端

支持独立的 OUI 客户端, 实现多 EAP 集中操作与监控.

- **架构独立**: 支持一个 OUI 对应多台 EAP, OUI 关闭不影响 EAP 运行.
- **权限与语言**: 支持账号权限管理与中英文消息.
- **信息查询**: 支持消息记录及设备, 物料, 错误信息查询.
- **异常干预**: 支持手动指令发送和异常处置.

### 14. 运维管理

#### 14.1 MC部署与版本管理

支持 EAP 的集中运维, 远程部署与版本全生命周期管理.

- **运行管控**: 支持 EAP 状态查询, Start/Stop/Restart, Code Version 查看, 日志查看.
- **部署升级**: 支持远程部署, 批量升级, 版本比对, 预约升级.
- **变更管理**: 支持 Pilot 验证, Rollback, 跨环境同步.

#### 14.2 日志、审计与监控

支持完善的日志管理, 监控告警与审计能力.

- **日志管理**: 支持 Log 分级, 按时间/大小切分, 定时备份和清理.
- **分析工具**: 支持问题追踪与日志分析工具.
- **监控告警**: 支持第三方监控接入与异常告警.
- **审计追踪**: 支持部署与操作审计.

### 15. 测试工具

#### 15.1 模拟器与测试环境

支持完整的模拟器与测试工具链, 覆盖开发到量产切换的全流程验证.

- **模拟回放**: 支持 SECS Log 回放, 设备模拟器, MES 模拟器, EAP 模拟器, SECS 测试工具.
- **测试验证**: 支持 NewType 模板开发, Lab Test, Fab Test, 回归测试和量产切换验证.

## FMB Function List

### 1. 系统

#### 1.1 看板设计

支持无需开发的可视化看板设计能力.

- **拖拉拽设计**: 系统提供丰富的组件, 无需开发, 通过拖拉拽方式设计界面.

#### 1.2 看板组件

支持丰富的看板组件库, 满足多样化展示需求.

- **组件库**: 提供 Chart, 表格, 图片, 数据源, 音频, 视频, 数据处理组件等 90+ 组件.

#### 1.3 播放管理

支持灵活的多看板播放策略.

- **轮播与分屏**: 支持多看板轮播, 可设定轮播时间, 支持分屏播放与单个播放.

#### 1.4 定时刷新

支持看板数据的定时自动刷新.

- **刷新机制**: 可设定刷新时间, 定时刷新看板数据, 最小单位 1 秒.

#### 1.5 3D仓库

支持以 3D 方式呈现仓库现场.

- **3D 展示**: 以 3D 方式设计仓库现场, 点击货架展示货架物料信息.

#### 1.6 表单功能

支持 Form 表单能力, 实现数据交互.

- **表单搜索**: 提供 Form 表单功能, 结合组件可实现搜索功能.

#### 1.7 动态事件

支持组件间联动的事件机制, 实现复杂应用场景.

- **事件联动**: 结合组件之间的传值和不同事件, 可实现事件监控等复杂应用场景.

#### 1.8 动画效果

支持多种动画效果, 提升看板展示体验.

- **动画支持**: 提供多种动画效果, 丰富看板体验.

#### 1.9 数据源管理

支持以简单配置方式对接业务系统数据.

- **数据库连接**: 通过简单配置方式连接业务系统数据库.

#### 1.10 多种数据源

支持多数据源并行配置.

- **多库配置**: 同一网络环境内可配置多个不同关系型数据库.

#### 1.11 看板数据管理

支持管理员在线配置看板数据获取逻辑.

- **SQL 配置**: 在管理员界面里配置 SQL 来获取数据, 不需额外连接其他开发工具.

#### 1.12 用户管理

支持多类型用户的分级管理.

- **用户分类**: 用户分为管理员和普通用户, 并区分可设计用户和只能查看的用户.

#### 1.13 权限管理

支持按看板组维度的访问权限控制.

- **权限分配**: 按看板组的维度, 给不同用户开放可访问权限.

#### 1.14 字体管理

支持自定义字体的扩展与使用.

- **自定义字体**: 可自定义添加字体, 从其他地方下载后可添加到系统, 设计看板时使用.

#### 1.15 附件管理

支持多种类型附件的统一管理.

- **附件上传下载**: 支持图片, 视频, 音频, Excel 文件等多种附件的上传下载管理.

#### 1.16 日志管理

支持系统运行与用户行为的日志管理.

- **日志记录**: 支持错误日志管理与用户登录管理.

### 2. 客制化

#### 2.1 设备状态监控

支持按用户现场布局定制的设备状态可视化监控.

- **Layout 定制**: 按照用户提供的厂区车间 Layout 定制设备 Layout, 按实际位置摆放设备.
- **状态展示**: 用不同颜色展示设备不同的状态.
- **详情查看**: 点击设备展示设备详细的信息.

#### 2.2 其他看板

支持 MES 相关业务的实时看板定制开发.

- **实时看板定制**: 可定制开发设备, 品质, 生产, 物料, 计划等 MES 相关实时看板.
