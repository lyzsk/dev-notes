# 平台功能
- 产品提供的控制模型：MA,WMA,EWMA,MPC; 支持用户开发自定义控制模型; 完全国产化，自主知识产权; 支持集群，具有高可用性; 支持Windows/Linux平台部署; 支持SQL Server/ Oracle/MySQL/Pg SQL/Vastbase G100/Open Guass等多种数据库; 支持多种消息总线，如Tibco RV/Raven Cast/H101/Web API/Rabbit MQ等，和其他系统通讯; 支持pilot，rework，special，runcard ，normal run等scenario; 软件升级以及新功能发布不影响系统使用，不停机; 消息并行处理，处理速度快; 易于通过客制化扩展自定义功能; 支持图形化workflow; 提供图形化的接口与其他系统集成; 支持workflow及sub workflow Debug; 图形组件快速定位; 图形组件多选/全选，复制，整体移动; 产品提供开发验证; 开发的流程间可嵌套，提高重用性; 提供测量数据收集及过滤功能; 提供收值项目和机台下货参数的配置; 产品提供常用函数如最小值、最大值、平均值、标准差等; 可自定义异常量测数据以及客制化检查流程，例如滤除低GOF (goodness-of-fit)量测资料不参与R2R反馈计算; 提供机台支持提供不同程度的前馈反馈控制器开发，如lot-level, batch-level, or wafer-level 前馈和后馈; 根据前值选择不同的生产recipe; 支持多步骤的同一Parameter同时控制; 计算模型版本控制; Request的数量及生命周期按不同控制器统计; Web UI，支持多浏览器访问; 登录支持接入OA验证; 支持SISO/MIMO系统; 分module R2R执行情况的监控; 查询 R2R计算执行失败履历;
# Litho Controller
- Litho CD控制模型：SISO EWMA Model; Litho CD FFFB模型：Lot级别反馈; Litho 支持Process机型：ASML，Canon，NIKON; Litho CD控制参数: Dose,Focus; ; Litho CD输出参数: CD ; 支持pilot run, rework run, normal run, special run, runcard run; 参数值最小调整阈值; tuning参数的变化量上限控制; tuning参数的上下限控制; Control Flag: ON，FIX(固定值模式)，OFF; 量测数据有效性检查; 支持固定值模式; 反馈有效期控制; 反馈有效批次控制; 按Lot Type反馈; 根据最近的run货值计算返工lot下货值; Thread 状态控制; FEM下货参数支持; OVL控制模型：MSISO EWMA; OVL FFFB模型：Lot级别反馈; 支持Process机型：AMSL/CANON/NIKON; 支持8、10 para线性参数; 支持HOPC/iHOPC 高阶参数; 支持pilot run, rework run, normal run, special run, runcard run; 参数值最小调整阈值; tuning参数的变化量上限控制; tuning参数的上下限控制; Control Flag: ON，FIX(固定值模式)，OFF; 量测数据有效性检查; wafer数据有效性检测; 支持固定值模式; 反馈有效期控制; 反馈有效批次控制; 按Lot Type反馈; 根据最近的run货值计算返工lot下货值; Thread 状态控制; Tool dedication检查; Simulate功能; 曝光记录手动补录; Chuck dedication; Wafer级别前层对准;
# CMP Controller
- 控制模型：MIMO MPC; FFFB模型：Lot 级别的前馈/后馈,Wafer 级别的前馈; 特别地，对于CMP集成量测支持Wafer级别反馈; 控制参数:研磨时间，研磨时间+分区压力; 输出参数:  研磨量，分区厚度; 支持pilot run, normal run, special/runcard,rework run功能; 参数值最小调整阈值; tuning参数的变化量绝对值上限; tuning参数的上下限; 调整参数值的截取(根据上下限或者变化量上下限); Control Flag: ON，FIX(固定值模式)，OFF; 下货参数值与实际下货值差异检查; 量测数据site有效性检查; 主从参数设置; 手工前量补录; 有效反馈的有效期控制， 控制绪的反馈值连续超出规定时间没有更新，此控制绪切换至pilot状态; 量测可根据wafer process run进行反馈; 自动侦测维修; 自动设置维修事件; 后量连续不合格，控制绪切换至pilot状态;
# ETCH Controller
- 控制模型：MIMO MPC; FFFB模型：Lot 级别的前馈/后馈,Wafer 级别的前馈; 控制参数: 蚀刻时间，ESC Chuck Temperatures，Gas Flows 等; 输出:  CD，Thickness，depth; 支持pilot run, normal run, special run，runcard run功能; 参数值最小调整阈值; tuning参数的变化量绝对值上限; tuning参数的上下限; 调整参数值的截取(根据上下限或者变化量上下限); Control Flag: ON，FIX(固定值模式)，OFF; tuning参数连续达到上下限的次数控制; 提供不同Input之间线性关系的约束 ; 量测数据site有效性检查; 支持基于单output 的多个zone的计算，; 有效反馈的有效期控制; 最新量测反馈控制; 支持主/从tuning 参数; 提供wafer level前馈时 缺少wafer的量测数据，自动补偿计算; 手动调整tuning parameter下货值侦测; 支持前值/后值量测来自于多个站点; 支持Hydra uniformity system（raw metro data and X/Y coordinates ）; 支持有关first wafer效应的处理;
# Furnace Controller
- 控制模型：MIMO MPC; FFFB模型：Batch 级别的反馈:; 支持Process机型：Kokusai furnace, Tel furnace; 控制参数： 各区的温度(usually 5 or 6, up to 10), deposition的时间或者ALD loop; 输出参数：各区的厚度(usually 5 or 6, up to 10); 支持pilot run, normal run, special run, runcard run scenarios; Zone的反馈资料识别; 根据必要Zone量测资料反馈; 有效反馈的有效期控制; 输出预测; 量测数据site有效性检查; 量测有效检查; 最新量测反馈控制; 量测超期处理; 参数值最小调整阈值; tuning参数的变化量绝对值上限; tuning参数的上下限; 调整参数值的截取(根据上下限或者变化量上下限); Control Flag: ON，FIX(固定值模式)，OFF; 下货tuning参数连续超限处理; 手动调整tuning parameter下货值侦测; 控制器Enable/Disable; 手动调整tuning parameter 参数的delta限制; Boat zone定义;
# WET Controller
- 控制模型：SISO EWMA; FFFB模型： Batch级别前馈，反馈; 支持1）根据前量厚度选择recipe，2）更新ETCH_TIME两种模式; 控制参数：ETCH_TIME; 输出参数:  Thickness; 支持pilot run,normal run, special run，runcard run; tuning参数的变化量绝对值上限; tuning参数的上下限; 调整参数值的截取(根据上下限或者变化量上下限); Control Flag: ON，FIX(固定值模式)，OFF; 量测数据site有效性检查; 量测数据wafer有效性检查; 量测数据lot有效性检查并做触发action; 有效反馈的有效期控制， 控制绪的反馈值连续超出规定时间没有更新，会hold住; 量测可根据wafer process run进行反馈;
# Thin Film Controller
- 控制模型：MIMO MPC; FFFB模型：Lot 级别的前馈/后馈,Wafer 级别的前馈; 控制参数: Dep Time,RF Power 等; 输出:  Thickness; 支持pilot run, normal run, special run，runcard run功能; 参数值最小调整阈值; tuning参数的变化量绝对值上限; tuning参数的上下限; 调整参数值的截取(根据上下限或者变化量上下限); Control Flag: ON，FIX(固定值模式)，OFF; tuning参数连续达到上下限的次数控制; 提供不同Input之间线性关系的约束 ; 量测数据site有效性检查; 支持基于单output 的多个zone的计算，; 有效反馈的有效期控制; 最新量测反馈控制; 支持主/从tuning 参数; 提供wafer level前馈时 缺少wafer的量测数据，自动补偿计算; 手动调整tuning parameter下货值侦测; 支持前值/后值量测来自于多个站点;
# Trim Controller
