# 缺陷数据查询模块
- 可按时间过滤筛选缺陷数据
- 数据过滤（支持产品、批次、晶圆、站点、配方等维度）
- 可按数据进行缺陷数据的过滤筛选
- 可按照原始数据的条件进行缺陷数据的筛选过滤
- 可设置仅呈现最终检测结果
- 可筛选过滤有照片的检测记录
- 可直接读取Klarf档案
- 可选择多种显示模式 (列表、行列定义)
- 可查询缺陷分类结果以及如DDC/DDP等指标
# 数据查询
- 可查询原始数据
- 用户可自行定义图形显示条件
- 支持汇出成Excel、CSV
# 多图显示模块
- 可自由调整显示的行列数量
- 支持缺陷群聚分析
- 可自由调整缺陷类别的显示颜色
- 可显示检测资讯
- 支持抽样分析
# 缺陷晶圆图的单图与叠图显示模块
- 可显示检测资讯
- 可调整缺陷晶圆图的显示细项条件
- 可放大或缩小缺陷晶圆图
- 可一键复制至剪贴板子
- 缺陷晶圆图中，缺陷显示的条件可自由调整
- 芯片资讯可选择是否显示
- 芯片边界可选择是否显示
- 可输出Klarf档案
- 可指定进行缺陷的检查拍照
- 自动缺陷抽样 (支持多种多重条件)
- 用户可在缺陷晶圆图中以任意形状选取缺陷
- 缺陷晶圆图可与缺陷数据互动显示
- 缺陷晶圆图可与统计图互动显示
- 统计图支持改变各种显示条件如X/Y轴条件、开关缺陷是否有分类或缺陷尺寸等
- 缺陷照片支持多种显示模式 (列表、行列定义、按条件排序等)
- 晶圆中支持按区域或Shot等进行缺陷的叠图
- 支持 CP测试结果与缺陷叠图
- 缺陷晶圆图的单图与叠图时支持区域划分与统计分析
- 可测量两个缺陷间的距离
# 新增缺陷分析
- 分别呈现前测、后测、前后测共同、后测新增等缺陷晶圆图
# 缺陷晶圆图的比较
- 可按后测增加、后测消失、前后测有差异等条件进行缺陷显示颜色的改变
# 缺陷分类模块
- 界面上支持缺陷缺陷的重新人工分类
# 绘图模块
- 支持各式趋势图、条形图、堆叠条形图、面积图、散点图、盒须图
# 缺陷照片浏览模块
- 支持树状缺陷照片浏览
- 可放大或缩小缺陷照片、可点击照片进行重新分类
- 支持将同一颗缺陷之不同照片堆栈
# 动态缺陷根源分析模块
- 用户可自行调整缺陷根源分析的缺陷位置误差范围
- 有缺陷根源分析的晶圆层级汇总
- 有缺陷根源分析的相关性汇总表
- 提供缺陷根源分析后的命中缺陷对照表
