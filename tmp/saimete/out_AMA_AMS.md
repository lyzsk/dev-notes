## AMA Function List

### 1. 管理后台

#### 1.1 业务建模

支持 AMA 自动化流程的可视化建模与配置, 覆盖流程、表单、连接与运行参数的定义.

- **流程模型**: 支持业务流程模型的设计与维护, 定义自动化派工 Workflow.
- **表单模型**: 支持流程表单模型的建模, 支撑流程中的人机交互界面.
- **连接服务配置**: 在建模阶段配置流程所需的外部连接服务.
- **流程配置**: 支持流程运行参数与触发条件的灵活配置.

#### 1.2 流程版本管理

支持流程的多版本发布与管理, 保障流程变更的可控与可追溯.

- **版本发布与管理**: 支持流程版本的发布、切换与历史版本管理, 确保生产流程平稳迭代.

#### 1.3 发起流程

支持多种方式发起自动化流程实例.

- **多模式触发**: 支持手动发起、定时触发与事件触发三种流程启动方式.

#### 1.4 查看流程

支持流程运行状态的实时监控.

- **流程实例监控**: 支持查看流程实例的执行进度、节点状态与执行结果.

#### 1.5 应用管理

支持 AMA 应用的创建与全生命周期管理.

- **应用创建与管理**: 支持自动化应用的创建、编辑、发布与管理.

#### 1.6 调度服务

支持流程与后台任务的定时调度执行.

- **流程定时触发**: 支持按预定时间周期定时触发流程执行.
- **常规任务**: 支持常规后台任务的调度与管理.

#### 1.7 连接服务

支持与外部数据源及系统接口的集成配置.

- **多类型连接**: 支持数据源、API、消息队列等多种外部连接的配置与管理.
- **策略与模板**: 支持连接策略与连接模板的定义与复用.

#### 1.8 通知服务

支持多渠道的消息通知触达.

- **多渠道通知**: 支持站内通知、邮箱、企业微信、短信等多种通知方式.

#### 1.9 Action 管理

支持流程节点的编辑与配置.

- **节点编辑与配置**: 支持流程 Action 节点的创建、编辑与运行参数配置.

#### 1.10 业务对象

支持流程运行所需业务对象的定义与管理.

- **内置与自定义对象**: 支持系统内置业务对象及用户自定义对象的创建与维护.

#### 1.11 系统管理

支持 AMA 平台的用户、权限与基础数据管理.

- **组织架构与角色管理**: 支持组织架构维护与用户角色管理.
- **菜单与权限管理**: 支持功能菜单与访问权限的细粒度配置.
- **基础字典**: 支持系统基础字典数据的维护.
- **公告管理**: 支持系统公告的发布与管理.
- **触发配置**: 支持流程触发规则的统一配置.
- **RTD同步**: 支持与 RTD 的数据同步, 保证派工规则与基础数据一致.

#### 1.12 运行监控

支持平台运行状态的全面监控与运维管理.

- **流程实例监控**: 支持全局流程实例的运行状态监控.
- **全局锁管理**: 支持全局资源锁的查看与管理, 避免并发冲突.
- **服务监控**: 支持系统各服务运行状态的实时监控.
- **在线用户**: 支持在线用户的查看与管理.
- **系统日志**: 支持系统运行日志的查询与追溯.

### 2. 前台服务

#### 2.1 首页

提供用户工作入口与数据概览.

- **功能导航与数据概览**: 支持功能入口导航与关键业务数据的概览展示.

#### 2.2 流程中心

支持用户流程任务的集中处理与流程发起.

- **待办任务**: 支持待办任务的集中展示与处理.
- **我的草稿**: 支持未提交流程草稿的保存与继续编辑.
- **流程追踪**: 支持查看我发起的、我处理的及抄送我的流程记录.
- **代理设置**: 支持流程处理代理人的设置, 保障任务不因人员缺席而中断.
- **发起流程**: 支持在前台直接发起流程.

#### 2.3 应用中心

支持 AMA 应用的统一展示与访问.

- **应用列表与详情**: 支持已发布应用的列表浏览与详情查看.

## AMS Function List

### 1. 用户管理

支持 AMS 用户账号、角色与权限的统一管理.

- **角色管理**: 支持系统角色的创建与维护.
- **角色权限**: 支持按角色配置功能访问权限.
- **账号管理**: 支持用户账号的创建、维护与停用.
- **用户组管理**: 支持用户组的定义与管理, 便于按群组分配通知与权限.

### 2. 设备信息

支持产线结构与设备基础信息的维护.

- **产线结构树**: 支持 Factory / System / Tool 等产线结构树信息的维护.
- **设备信息管理**: 支持设备 (Tool) 基础信息的录入与管理.

### 3. 系统管理

支持系统基础数据字典的维护.

- **数据字典**: 支持系统数据字典的定义与维护.

### 4. 系统设置

支持邮件与报警看板等系统级参数配置.

- **邮件设置**: 支持 SMTP 邮件发送参数的配置.
- **邮件模板设置**: 支持 Alarm 通知邮件模板的自定义配置.
- **报警看板设置**: 支持报警看板显示内容与刷新参数的自定义配置.
- **通用设置**: 支持系统通用参数的统一配置.

### 5. 基础配置

支持报警级别、规则与代码的核心配置管理.

- **报警级别管理**: 支持 Alarm 严重等级 (Severity) 的定义与管理.
- **报警规则设定**: 支持基于工厂、系统、设备、Alarm Code 等维度的报警规则配置, 控制警报的触发、屏蔽与动作.
- **报警统计配置**: 支持报警统计口径与维度的自定义配置.
- **报警代码维护**: 支持 Alarm Code 及其描述信息的维护.

### 6. 手动操作

支持警报的人工处置与转单管理.

- **状态切换**: 支持人工对 Alarm 进行状态切换 (如响应、关闭、终止).
- **报警转单记录**: 支持报警转单的记录与查询, 追溯处置过程.
- **报警作业**: 支持报警相关作业的人工执行与管理.

### 7. 查询统计

支持警报历史与动作记录的多维度查询统计.

- **报警历史记录**: 支持按多条件组合查询 Alarm 历史记录.
- **报警次数统计**: 支持按维度统计 Alarm 发生次数, 辅助问题分析.
- **报警动作记录**: 支持报警触发动作 (如通知、联动操作) 的记录查询与追溯.

### 8. 看板管理

提供实时可视化的报警态势感知能力.

- **报警看板**: 支持报警信息的实时看板展示, 直观呈现当前警报分布与状态.

### 9. 日志管理

支持系统运行日志的查询与审计.

- **日志查询**: 支持系统日志的检索与查询, 便于问题排查与审计追溯.

### 10. 定制功能

支持与企业现有基础设施的定制集成.

- **第三方通知平台**: 支持发送企业微信、飞书、短信等第三方通知平台, 实现多渠道告警触达.
- **SSO集成**: 支持 SSO 登录 (统一登录) 与账号验证, 与企业身份认证体系无缝对接.
