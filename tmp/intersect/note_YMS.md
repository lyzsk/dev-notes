### YMS

#### 仅上扬独有的功能(未纳入交集)
- 脚本管理(新建、打开、保存、另存为、关闭脚本)
- 系统帮助(使用手册、更新日志、版本号查看)
- 退出系统保护(退出时提示保存)
- 用户信息维护、用户组维护、功能维护(同步/启停)、功能包维护、公共脚本访问权限管理
- 产品光罩 (Reticle) 布局定义管理
- CP 数据产品定义管理、WAT 数据 SITE 定义管理、Zone 布局定义维护
- DSA 定义维护(By FAB、产品、Layer, EQP 设置 Tolerance)
- 缺陷分类定义、Cluster 定义、Repeat 定义、跨 FAB 缺陷分类映射定义
- 多数据源接入(CP, WAT, FT, WIP, Inline 等数据源)
- 数据分组(自定义分组、条件式分组)
- Map 分析: WAT Map (5 Site / 9 Site) 绘制、单片/多片叠图与多 Wafer Map 合并、Bin 统计分析与分布图表(柱状图)、Parameter 统计分析与直方图、Sort Map 颜色梯度(热力图)、Ink 操作与 Text Map 导出
- Map 样式: Wafer Map 导出 Excel/PowerPoint、复制到剪贴板、鼠标悬停提示、缩放旋转、Die / Site / Reticle Level 视图
- 双因子变异数分析 (Two Way ANOVA)
- 均匀累积分布图 (CDF)
- 图表交互操作(缩放、复制图表到剪贴板)、图表导出至 Excel/PowerPoint
- 工作流流程决策(条件分支走向判断)
- 报告共享文件夹分发机制(见 ⚠️ 说明)

- 表差异比对

- 插件集成: Excel 宏 (Macro)、R 语言、Python
- 报表增强: Excel 报表多表合并与 Index Sheet 自动链接、PPT 多章节/多图合并分页、邮件内含 HTML 报表
- 运维: 计划任务性能监控、系统运行日志、功能使用监控、License 管理、系统健康状况查询

#### 与原文不一致处(⚠️ 标记说明)
- 4.1 Map类型支持: 上扬原文为"支持查看良率 Map、参数 Map, WAT Map", 赛美特未提及 WAT Map, 交集仅保留两方共有的良率 Map 与参数 Map, 故删去"WAT Map".
- 4.1 光罩叠加: 上扬原文为"支持 Bin Map / Sort Map / WAT Map 和光罩叠加分析", 赛美特原文为"CP Map & 光罩版图", 两方 Map 类型口径不同, 合并改写为"支持 Wafer Map 与光罩版图叠加分析".
- 8.13 外观定制: 上扬原文为"支持设置图表标题、图例、轴距、步距、颜色", 赛美特原文为"支持接口样式、图型颜色、字体、大小等个性化配置", 两方表述维度不同, 取交集(颜色、字体等外观个性化)合并改写.
- 10.1 多格式导出: 上扬为"Excel, PowerPoint, PDF", 赛美特为"HTML, Excel (2003, 2007), CSV, PowerPoint (2003, 2007)", 两方格式清单不同, 交集仅含 Excel 与 PowerPoint, 故删去 PDF(上扬独有)与 HTML/CSV(赛美特独有).
- 10.1 报告分发: 上扬原文为"报告生成后, 支持发送邮件、导出至共享文件夹机制", 共享文件夹为上扬独有; 赛美特邮件可含 HTML 报表内容为赛美特独有, 交集仅保留"发送邮件".
- 其他表述差异(未改写, 引用一方原文): 1.1 语言设置引用上扬"中文、英文", 赛美特额外支持繁体中文; 2.1 文件导入引用上扬"Excel, CSV", 赛美特额外兼容 Txt 格式; 7.1 T 检验交集仅含单样本 T 检验, 赛美特另有双样本、配对样本 T 检验; 8.13 辅助线引用上扬"规格线、控制线", 赛美特对应表述为"规格线(含动态规格线)、常数线".

#### 原文来源选择说明
- 交集条目绝大多数引用上扬原文, 原因是上扬的功能清单粒度更细、描述更具体(如统计方法明确列出 Pearson/Spearman、汇总统计明确列出指标清单、工作流明确列出编排操作), 而赛美特对应条目多为一句话概括式描述. 仅在语言、权限、排程等少数两方粒度相当的条目中, 同样优先采用了上扬更精确的表述. 赛美特原文仅在总述层面(如 MAP 分析总述)被部分借鉴.
