## RMS Function List

### 1. Recipe 管理

#### 1.1 Recipe 管控与比对

支持 Recipe/Sequence 的管控范围设定, 多方式比对, 参数规范模板复用及校验异常处理, 确保生产配方的准确性与一致性.

- **管控列表与 ByPass**: 支持设定 Recipe/Sequence 管控列表, 默认纳入管控;支持设定 ByPass Recipe, 此类 Recipe 不进行比对;支持设置 SubRecipe 是否进行卡控.
- **多种比对方式**: 支持 Recipe FullBody 与 Recipe Parameter 等形式的比较. ⚠️
- **设备间比对**: 支持设备间程序差异比较, 支持相同程序批量比较多个设备之间的差异, 支持两个设备批量比较多个程序之间的差异.
- **版本比对**: 支持不同版本程序比较.
- **Spec 模板复用**: 支持多设备共用参数规范模板, 参数规范可保存成模板. ⚠️
- **关键参数管理**: 支持设置关键参数 (一方支持关键参数高亮显示, 另一方支持按机型设定 Key Param 的 Spec 范围). ⚠️
- **校验与异常处理**: 支持 Recipe/Sequence 校验, 以及校验失败邮件通知等相关的异常处理流程.

#### 1.2 版本管理与批量作业

支持 Recipe 多版本与 Active 版本管控, 系统与设备间的上传下载及批量作业, 提升批量运维效率.

- **多版本管控**: 支持 Recipe 多版本管控与 Active 版本管控, 系统内可保存多个 Recipe 版本.
- **上传下载与批量作业**: 支持系统与设备之间上传与下载 Recipe/Sequence, 支持批量上传, 批量下载及批量 Active. ⚠️
- **Recipe 共享**: 支持同型号设备之间共享 Recipe. ⚠️

#### 1.3 履历查询

支持 Recipe/Sequence 版本变更与校验履历的完整追溯.

- **变更履历查询**: 支持查询 Recipe/Sequence Version 的变更履历及 Parameter SPEC 修改履历. ⚠️
- **校验履历查询**: 支持查询 Recipe(Sequence) 的 EAP Validation 履历.

### 2. EC 管理

#### 2.1 EC 管理

支持设备常量 (EC) 的参数规范管控, 多设备共享及历史追溯.

- **常量定义**: 支持新建设备常量参数, 并编辑参数规范, 支持设备常量的 Absolute, ByRange, ByTolerance, ByList 比较类型.
- **多设备共享**: 支持多设备 (同类型设备) 之间共享, 复制设备常量设定. ⚠️
- **变更履历**: 支持查询设备常量的设定变更履历.
- **校验履历**: 支持查询设备常量的 EAP Validation 履历.

### 3. 签核系统集成

#### 3.1 签核与审批

支持与外部签核系统整合的审批管理与关键节点邮件通知, 确保配方变更的合规审批.

- **外部系统整合**: 支持与外部签核系统整合.
- **审批操作**: 支持 Approve/Reject 程序.
- **批量签核**: 支持批量签核. ⚠️
- **通知时机设定**: 支持设定邮件通知时机点, 如 Recipe Validation Fail, Set Active 等.

## RCM Function List

### 1. 远程控制

#### 1.1 远程操作

支持多客户端环境下的远程机台操作与操作权管控, 实现安全可控的远程作业.

- **远程控制**: 支持正常的远程操作, 包括双荧幕的操作.
- **并发策略**: 允许多个客户端查看同一设备, 同时只有一个客户端可以控制它.
- **鼠标合一**: 自动将远端与近端的鼠标合二为一, 避免两个鼠标出现在同一个画面的困扰.

#### 1.2 记录与审计

支持操作过程的影音记录与操作记录保存, 满足审计追溯需求.

- **影音与日志**: 支持录像, LOG 记录, 截图功能.
- **操作记录**: 支持操作记录保存, 包括人员操作 RCM 的记录, 保存时长取决于存储空间.

### 2. 监控画面管理

#### 2.1 画面布局与设备检索

支持监控画面布局的自定义配置与设备快速检索定位.

- **宫格布局自定义**: 用户可自定义宫格页面, 支持 1x1, 2x2, 3x3 等多种规格.
- **模糊查询**: 支持机台 ID 模糊查询.

### 3. 账号与权限

#### 3.1 登录与权限

支持账号密码验证登录与基于权限的机台访问控制.

- **账密验证**: 输入账号密码, 通过后端接口完成身份验证.
- **权限化加载**: 按照设定权限加载用户可见的机台.
