## YMS Function List

### 1. 系统管理

#### 1.1 语言设置

支持系统界面的多语言切换.

- **多语言切换**: 支持中文、英文界面语言切换.

#### 1.2 权限管理

提供功能粒度的权限管控能力.

- **访问权限控制**: 支持设置用户或用户组对功能、功能包的访问权限.

#### 1.3 排程管理

支持分析任务的自动化排程执行.

- **自动排程**: 支持设置排程任务, 自动运行脚本并输出报告.

### 2. 数据导入

#### 2.1 文件导入

支持本地文件的便捷数据导入.

- **文件批量导入**: 支持通过 Excel, CSV 格式文件导入数据, 支持多文件批量导入.

#### 2.2 DB 导入

支持多条件组合的数据库数据导入.

- **条件组合筛选**: 支持设置不同的条件 / 时间范围组合筛选.
- **时间规则查询**: 支持日历时间查询, 支持自定义时间规则.
- **通配符查询**: 查询条件支持通配符.

#### 2.3 SQL 查询

支持自定义 SQL 语句的灵活数据导入.

- **SQL直连导入**: 支持通过编写 SQL 语句实现从 DB 中导入数据.

### 3. 数据处理

#### 3.1 列转行

支持数据结构的行列转换.

- **列转行**: 支持数据的行列转换, 如列转行.

#### 3.2 行转列

支持数据结构的行列转换.

- **行转列**: 支持数据的行列转换, 如行转列.

#### 3.3 数据连接

支持多数据集之间的合并与运算.

- **数据合并运算**: 支持不同类型数据的左右合并、上下合并、相减.

#### 3.4 补值

支持缺失数据的灵活补值处理.

- **缺失值补值**: 支持统计值补值与自定义补值.

#### 3.5 数据筛选

支持规则化与统计化的数据过滤.

- **数据过滤**: 支持自定义规则过滤数据, 支持使用统计方法筛选数据.

#### 3.6 透视表

支持数据的聚合透视分析.

- **数据透视**: 支持行列聚合、数据透视功能.

#### 3.7 数据导出

支持处理结果的多格式导出.

- **多格式导出**: 支持数据导出为 Excel, CSV 格式.

### 4. MAP 分析

#### 4.1 CP/Sort Map

提供晶圆图谱 (Wafer Map) 分析能力, 检视各种检测数值在 Map 上的分布结果.

- **Map类型支持**: 支持查看良率 Map、参数 Map ⚠️
- **Zone分析**: 支持 Wafer Zone 分析, 支持用户自定义分区计算 Yield, Parameter.
- **光罩叠加**: 支持 Wafer Map 与光罩版图叠加分析 ⚠️

### 5. 基础分析

#### 5.1 相关性分析

支持变量间的相关性量化分析.

- **Correlation分析**: 支持 Correlation (Pearson, Spearman) 分析功能.

#### 5.2 数据汇总

支持全面的描述性汇总统计.

- **汇总统计**: 支持数据汇总统计, 包括均值、标准差、最大值、最小值、求和、个数、四分位数、百分位、IQR、偏度、峰度.

#### 5.3 单因子变异数分析

支持单因素方差分析.

- **One Way ANOVA**: 支持单因素方差分析 One Way ANOVA 分析功能.

#### 5.4 单因子变异数分析 (重复测量)

支持重复测量场景的单因素方差分析.

- **重复测量ANOVA**: 支持单因素方差分析 One Way ANOVA (重复测量) 分析功能.

#### 5.5 正态性检验

支持数据分布的正态性验证.

- **Normality Test**: 支持 Normality Test 分析功能.

#### 5.6 共通性分析

支持多维度共通性 (Commonality) 分析.

- **共通性分析**: 支持共通性分析功能, 支持根据不同的字段排序优先级.

### 6. 多变量分析

#### 6.1 主成分分析

支持主成分降维分析.

- **PCA分析**: 支持 Principle Component Analysis (PCA) (with loadings, scores, scree plot, scatter plot) 分析.

#### 6.2 偏最小二乘回归

支持偏最小二乘回归建模分析.

- **PLS回归**: 支持 PLS Regression (with X/Y loadings, scores, and VIP) 模型分析.

#### 6.3 多元回归

支持多元线性回归建模分析.

- **多元线性回归**: 支持 Multiple Linear Regression 分析功能, 支持设置多项式变量.

#### 6.4 决策树

支持决策树模型分析.

- **Decision Tree**: 支持 Decision Tree 分析.

### 7. 假设检验

#### 7.1 T 检验

支持单样本均值假设检验.

- **One Sample T-Test**: 支持 One Sample T-Test 分析功能.

#### 7.2 F 检验

支持联合假设检验.

- **F Test**: 支持 F Test (联合假设检验) 分析功能.

#### 7.3 两样本集成检验 (PCRB)

支持工艺变更前后差异的定量评估.

- **PCRB检验**: 支持辅助用户对工艺变更前后的样本间差异进行定量分析.

### 8. 图形绘制

#### 8.1 趋势图

支持多形态的趋势图绘制.

- **趋势图绘制**: 支持趋势图, 包括 Line, Line+Point, Bar, BarStacked.

#### 8.2 时序图

支持时间序列数据的可视化.

- **时序图绘制**: 支持时序图.

#### 8.3 盒须图

支持分布特征的盒须图展示.

- **盒须图绘制**: 支持盒须图, 盒须图支持显示均值、标准差、四分位数等统计值.

#### 8.4 散点图

支持变量关系的散点图展示.

- **散点图绘制**: 支持散点图.

#### 8.5 矩阵散点图

支持多变量两两关系的矩阵化展示.

- **矩阵散点图绘制**: 支持矩阵散点图.

#### 8.6 3D 散点图

支持三维空间的散点数据展示.

- **3D散点图绘制**: 支持 3D 散点图.

#### 8.7 正态概率分布图

支持正态性评估的概率图展示.

- **NPP绘制**: 支持正态概率分布图 (NPP).

#### 8.8 帕累托图

支持关键少数因素的帕累托分析展示.

- **帕累托图绘制**: 支持帕累托图.

#### 8.9 直方图

支持数据分布的直方图展示.

- **直方图绘制**: 支持直方图.

#### 8.10 饼图

支持占比关系的饼图展示.

- **饼图绘制**: 支持饼图.

#### 8.11 曲面图

支持三维曲面数据的展示.

- **曲面图绘制**: 支持曲面图.

#### 8.12 等高线图

支持平面等值线数据的展示.

- **等高线图绘制**: 支持等高线图.

#### 8.13 样式功能

提供图表分组、布局与外观的定制能力.

- **分组分图**: 支持分组设置, 同一图内以不同颜色区分绘制不同组别的数据;支持分图设置, 同类型图表多图显示在同一窗口.
- **外观定制**: 支持图表颜色、字体、大小等外观个性化设置 ⚠️
- **辅助线**: 支持在图表上添加规格线、控制线.

### 9. 工作流

#### 9.1 流程编排

支持可视化工作流 (Workflow) 的编排与执行.

- **模块管理**: 支持新增、编辑、删除功能模块.
- **布局编排**: 支持拖拽模块布局, 支持一键垂直布局、水平布局.
- **数据承接**: 支持两模块之间连线, 承接数据.
- **一键执行**: 支持流程一键执行.

### 10. 报告

#### 10.1 报告设置

支持报告的自定义生成、自动导出与分发.

- **报告定制**: 支持自定义报告版式布局、报告导出设定.
- **多格式导出**: 支持把分析结果导出到 Excel, PowerPoint 文件中 ⚠️
- **自动报告**: 支持自动报告导出功能, 系统可以自动运行指定的分析模板生成并导出报告.
- **报告分发**: 报告生成后, 支持发送邮件 ⚠️
