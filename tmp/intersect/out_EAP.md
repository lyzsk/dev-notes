## EAP Function List

### 1. 设备接入与状态管理

#### 1.1 设备通讯与外设整合

支持标准通讯协议与各类外设的设备接入, 实现单台 EAP 对多设备的控制与信息交互.

- **标准协议**: 支持 SECS-I, HSMS, SECS-II, GEM 等半导体标准通讯.
- **外设整合**: 与机台、BarCode 扫码枪、SmartTag/SMIF、RFID 等整合, 实现对设备的控制与信息交互.
- **多设备接入**: 支持单台 EAP 连接多个设备, 如 Inline 机台、SMIF、Smart Tag、RFID Reader.
- **通讯可靠性**: 支持设备重连或者 Offline 切换到 Online 的自动初始化.

#### 1.2 设备状态监测与控制

支持设备通讯状态、控制状态的监测、同步与切换.

- **状态监测**: 设备通讯状态监测, 至少包含通讯断开 (Disconnect)、设备离线 (Offline)、设备本地控制 (Online-Local)、设备远程控制 (Online-Remote), 同步设备状态.
- **状态检查**: 设备控制状态与工艺状态检查, 同步设备状态变化.
- **状态切换**: 支持切换设备 Offline/Online 状态.

### 2. 跑货流程控制

#### 2.1 Scenario 配置与流程控制

支持 EAP Scenario 的可视化配置与灵活的流程控制, 并提供多类型设备框架/模板.

- **图形化配置**: 支持 EAP Scenario 的图形化配置, 显示和修改.
- **流程配置**: 支持异常流程, 动态事件配置, Inline 多设备配置. ⚠️
- **作业粒度控制**: 支持多种下 Job 方式, By Lot, Batch, Wafer 等. ⚠️
- **设备框架/模板**: 支持 Fixed Buffer, Internal Buffer, Sorter, FOUP Clean 等多类型 EAP 框架/模板. ⚠️

#### 2.2 工艺校验

支持工艺前的 Mapping 与 Recipe 校验, 保障跑货准确性.

- **Mapping 检查**: 支持工艺前的 Mapping 检查, 区分数量校验与位置校验, 可通过配置文件进行配置.
- **Recipe 校验**: 支持工艺 Recipe 一致性校验.
- **晶圆状态监控**: 支持工艺时晶圆工艺状态检查, 及时上报每片晶圆工艺状态.

#### 2.3 Load Port 与载具控制

支持 Load Port 载入载出控制与多类型载具适配.

- **Load Port 控制**: 支持控制 Load Port 载入载出, 同步 Load Port 状态变化.
- **载具类型适配**: 支持 FOUP/OpenCassette 等载具类型的控制与安全生产流程. ⚠️

#### 2.4 异常处理与警报管理

支持安全的异常处理恢复流程与设备警报的收集管理.

- **异常处理与恢复**: 支持安全且便于处理的异常处理流程, 支持流程异常中断的原地恢复.
- **异常处置**: 校验不匹配时支持 Hold Lot, 报警等处理. ⚠️
- **警报管理**: 支持实时设备警报收集、警报过滤、分级管理, 实现邮件通知、Hold Lot 等功能.

### 3. 系统集成与数据上报

#### 3.1 EAP 与 MES 整合

支持与 MES 的深度整合, 覆盖信息交互、匹配验证、作业执行与设备状态同步等核心业务.

- **信息获取与上报**: 支持请求 MES 获取 Carrier 信息、Lot 信息、设备信息、Batch 信息等, 并支持上报上述信息至 MES.
- **EDC 量测数据**: 支持请求 MES 获取 EDC 量测信息与上报设备量测数据.
- **匹配验证**: 支持请求 MES 验证 Lot 与设备是否匹配.
- **过程信息上报**: 支持上报 MES Process Start 与 Process End 时间及信息.
- **作业动作执行**:
    - 支持请求 MES 执行 Job In、Job Out、Cancel Job In、Hold Lot 等动作.
    - 支持请求 MES 执行 Batch Job In、Batch Job Out.
- **设备状态同步**: 支持 MES 控制设备 Offline/Online 状态, EAP 需同步更新设备状态至 MES.

#### 3.2 EAP 与 RMS 整合

支持与 RMS 系统的接口整合, 实现 Recipe 的双向传输与校验.

- **Recipe 解析上传**: 支持对设备 Recipe Parameter, Recipe Body 的解析并上传至 RMS 系统.
- **Recipe 抓取**: 支持 RMS 系统请求抓取设备 Recipe/Sequence.
- **双向传输**: 支持设备 Recipe 上传至 RMS 系统与 RMS 系统中 Recipe 下载至设备.
- **Recipe 校验**: 支持 RMS 对设备 Recipe 的比对与一致性校验. ⚠️

#### 3.3 数据采集与上报

支持设备数据与量测数据的采集、处理与向上位系统上报.

- **采集内容**: 支持设备事件, 报警, 状态, 常量, 工艺参数等数据采集.
- **数据处理配置**: 支持量测/采集数据的过滤、格式转换与采集配置. ⚠️
- **数据上报**: 支持向 FDC, MES 等系统上报. ⚠️

#### 3.4 上位系统集成与接口方式

支持与各类上位系统集成, 接口方式灵活多样.

- **集成范围**: 支持与 MES, RMS, APC, FDC 等系统的整合.
- **接口方式**: 支持 Message Bus 消息中间件及 WebService/Web API 等接口方式, 实现与周边系统的消息通讯. ⚠️

### 4. EAP UI / 客户端

#### 4.1 EAP UI / OUI

支持图形化操作界面, 实现多机台聚合、权限管控与消息记录查询.

- **状态显示**: 支持简洁清晰的设备状态显示.
- **多机台聚合**: 支持一个 UI 对应多台 EAP, UI 关闭不影响 EAP 正常运行, 支持查询 UI 对应 EAP 信息.
- **权限管控**: 支持账号权限配置与管控. ⚠️
- **消息记录**: 支持 UI 消息记录查询. ⚠️

### 5. 日志与运维

#### 5.1 EAP 日志

支持分类清晰、可灵活配置的日志机制, 便于问题定位.

- **日志分类**: 提供详细灵活且分门别类的日志, 包括但不限于 SECSLog, EventLog, MESLog, RMSLog, UILog, ErrorLog, StatusLog, AlarmLog 等.
- **日志文件管理**: 日志文件可以根据时间和大小为每台设备创建. ⚠️
- **问题定位**: 提供根据日志进行问题定位与解决的方案.

#### 5.2 运维监控与部署升级

支持统一的运维监控与灵活的部署升级能力.

- **运维监控**: 提供统一的运维监控平台, 支持异常邮件通知.
- **版本回退**: 支持版本回退.
- **更新升级**: 支持单台更新和批量更新.

#### 5.3 模拟器与测试工具

支持完整的模拟器与测试环境, 便于开发测试与运维.

- **设备与 MES 模拟器**: 提供设备模拟器与 MES 模拟器, 便于本地测试.
- **MES 消息模拟**: 支持根据 MESLog 模拟 MES 进行消息收发, 复现日志, 便于接口开发测试与运维.
- **EAP 模拟测试**: 支持模拟 EAP 进行 SECS 测试, 便于设备测试与运维.
- **日志回放测试**: 支持根据 SECSLog 模拟机台进行跑货测试, 便于设备测试开发与运维.
- **测试环境**: 提供完整的测试环境, 进行测试与开发.
