## SPC Function List

### 1. 数据采集与基础设置

#### 1.1 数据采集

支持多源生产数据的采集.

- **多源数据采集**: 支持工艺设备数据, 厂务数据等多种生产数据源的采集. ⚠️

#### 1.2 Chart 定义与建模

支持灵活的 Chart 组织, 建模与管控配置.

- **分组管理**: 支持以分组 (树状 Folder 或 Channel 分组/子组) 的方式分类组织管理 Chart. ⚠️
- **Context 过滤定义**: SPC Chart 可根据 Context Key 自定义过滤条件, 如 [产品]+[工艺路线]+[加工 Step]+[加工设备]+[Chamber]+[EDC Plan ID] 等;母 Chart (Group Chart), 子 Chart (Subgroup Chart) 的进点过滤条件使用 Context Key 表达式定义, 支持正向和反向 (排除名单) 过滤.
- **批量建模**: 支持 Excel 模板导入 / 导出建模数据.
- **属性继承**: 支持参数、Channel(数据采集维度)、Chart 三层建模数据的基本属性继承.
- **自动 Sub Chart 管控**: 支持根据维度/Context 值 (如工艺设备, Chamber, 产品, Step 等) 自动建立 Sub Chart/CKC 进行管控. ⚠️
- **Chart 分类属性**: 支持 K/NK/NC 属性, 用于区分关键、非关键、不管控等 Chart 类型.

#### 1.3 控制限与判异 Action

支持多种控制限及违规后的告警与联动处理.

- **多种控制限**: 支持 Control/Spec/Warning Line, Target, Center Line 等多种控制限. ⚠️
- **上下限判异**: 支持 OOS/OOC/OOW 等上下限判异. ⚠️
- **违规告警邮件**: 支持违规告警邮件发送, 可根据违规类型 (OOS, OOC, OOW 等) 单独设置.
- **OCAP 联动**: 异常数据可链接到 OCAP 系统, 并支持从异常点跳转至 OCAP 处理页. ⚠️

### 2. 规则与计算

#### 2.1 判异规则

支持标准化判异规则与灵活的自定义配置.

- **标准规则集**: 管制图的检查支持标准 WE Rule.
- **自定义规则配置**: 支持自定义方式配置 Rule, 如连续 n 点上升的 n 可配置.

#### 2.2 计算型参数 (Calc Parameter)

支持基于公式的计算型参数与丰富的计算函数.

- **多公式配置**: 支持计算型参数设定, 一个参数可配置多个计算公式.
- **计算函数**: 支持丰富的计算函数 (如选片选点函数, Round, Cos, Sin, Tan, Log 等), 可基于这些函数自定义公式, 支持多个参数的复杂运算.

### 3. 图表分析

#### 3.1 图表类型与显示

支持多类型图表与精细化的显示配置.

- **计量型图表 (Variables Charts)**: Xbar/X Chart, Range Chart, Sigma Chart (Standard Deviation), Moving Range Chart, Moving Sigma Chart, Moving Average Chart. ⚠️
- **计数型图表 (Attributive Charts)**: C Chart (不符合项), NP Chart (不合格数量), P Chart (不合格率), U Chart (平均不合格数).
- **分布类图表**: 支持正态分布图 (NormalProb), 箱线图 (Boxplot) 等分布类图表. ⚠️
- **X 轴标签**: 支持设置 X 轴 Label 默认显示信息 (时间、UNIT_ID, LOT_ID 等), 以及默认显示数量、倾斜角度等.
- **界限显示**: 支持显示历史或当前 Chart 上配置的规格, 控制界限, 显示方式可配置.
- **统计指标显示**: Chart 显示内容包括 Max, Min, Mean, 标准偏差, Total Count, OOC%, OOS%, Cp, Cpk 等统计指标. ⚠️
- **数据点详情**: 支持显示数据点详细信息, 包含数据采集的主数据与辅助信息.
- **数据点样式区分**: 支持正常点、异常点、备注点等不同样式显示.

#### 3.2 数据点操作

支持数据点的备注, 过滤, 隐藏及过滤后重算.

- **备注管理**: 支持对数据点添加备注, 并查看该点过往所有备注记录.
- **数据点过滤与隐藏**: 支持手动过滤, 隐藏 (Disable) 数据点, 并支持批量处理. ⚠️
- **过滤重算**: 隐藏点或过滤数据点后, 按照过滤后数据重新计算 Cpk, Avg 等相关统计数据.
- **特殊点过滤**: 支持按重量, 离散点等条件过滤 (排除) 特殊数据点. ⚠️
- **控制限计算**: 支持使用选定数据的 N 倍 Sigma 计算新的 Control Limit, 数据筛选条件可使用时间范围或数据点个数, 几倍 Sigma 支持可配置.

#### 3.3 图表样式管理

支持图表颜色与样式的自定义.

- **颜色管理**: 支持自定义不同控制线的颜色及数据点在不同状态下的颜色 (如 OOC 点的颜色).

#### 3.4 查询与收藏

支持 Chart 检索, 常用 Chart 收藏及批量操作.

- **Chart 搜索**: 支持使用 Chart 的属性搜索 Chart, 如 ChartName, ParameterName, Context 值, SPC Rule.
- **书签与收藏**: 支持将常用 Chart (及搜索/筛选条件) 保存为书签/收藏夹, 方便查找和查看. ⚠️
- **批量查看与导出**: 支持批量打开 (查看) 及批量导出 Chart. ⚠️

### 4. 统计与报表

#### 4.1 指标统计与报表

支持 Cpk 关键指标统计与周期性报表.

- **关键指标计算**: 支持 Cpk(d2), Cpk(MEAN), Ppk 等关键性指标计算, 统计哪些指标可配置.
- **定期统计**: 支持以 SPC Chart 为对象按日, 周, 月, 季等周期定期统计, 统计周期可配置;支持用户自定义查询 Chart 的过滤条件 (如 by Product, by Module, by Tool).
- **违规统计**: 支持违反 Alarm Rule 的 Count 及 Ratio 统计.
- **数据过滤**: 支持 OOS, OOC 等过滤功能, 被过滤 (排除) 的数据默认不参与 Cpk 统计. ⚠️

#### 4.2 数据导出

支持 Chart 数据与报表的文件导出.

- **格式导出**: 支持导出 Chart 与原始数据及报表, 导出格式支持 Excel 等常用文件格式. ⚠️
