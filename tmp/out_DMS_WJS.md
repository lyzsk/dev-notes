## DMS Function List

### 1. 缺陷分析

#### 1.1 数据查询

支持多维度、多条件的缺陷数据查询与高级筛选, 快速定位目标 Lot/Wafer 的 Defect 数据.

- **时间维度查询**: 支持相对时间与绝对时间两种时间模式查询.
- **快速筛选**: 支持按 Product, Lot, Wafer, EQP, Recipe 等不同条件快速筛选.
- **高级过滤**: 支持按 Defect Density, Scan Area, Defect Count, Defective Die Count, Has Images, Has Cluster, Has Repeat, Last Scan Only 等条件进行高级过滤.

#### 1.2 DSA (Defect Source Analysis)

支持缺陷根源分析运算, 并可通过 Layer 过滤与自定义配置灵活控制分析范围.

- **Cluster 分析**: DSA 支持 Cluster 分析计算.
- **Layer 过滤**: DSA 分析运算支持按照 Layer 进行过滤设置, 支持自定义选择前层参与运算.
- **Ignore Layer**: 支持用户自定义 Ignore Layer, 指定不参与 DSA 分析的 Layer.

#### 1.3 Wafer Map 分析

支持 Defect Wafer Map 的显示、交互、框选、叠图与对比分析, 并可与图像、图表及 Review 机台联动.

- **Map 显示**: 支持 Defect Wafer Map 显示及 Map 信息查看, 显示必要的 Defect 信息; 支持 Lot, Wafer 及 Layer 级别的 Map 分析; 支持 Defect Map By Class 查看.
- **显示控制**: 支持用户设定 Map 显示参数 (标题、大小、颜色等); 支持 Defect Size 调整; 支持快速过滤 (Show/Hide Defect Clusters, Show Adder Defect Only, Show Images Only, Show Repeat Only, Manual Classified Only).
- **图像联动**: Map 上的 Defect 可以链接到对应的 Defect 图像; 支持 Map, Chart, Data Set 交互联动.
- **备注管理**: 支持添加、删除 Defect 备注.
- **Repeater Analysis**: 支持重复缺陷分析, 通过叠加缺陷识别重复的 Defect, 支持 Die 及 Reticle 两种分析模式.
- **框选操作**: 支持矩形、自定义框选过滤 Defect; 支持 Select Filter.
- **Send to Review**: 支持在 Map 上框选 Defect, 并将结果 Send to Review 机台.
- **Cluster 重算**: 支持用户自定义条件, 重新计算、显示以及修改 Cluster 信息.
- **导出功能**: 支持在 Map 上导出 KLARF, Txt Map.
- **Gallery Map**: 支持 Gallery Map 自定义布局及透视分析.
- **对比分析**: 支持两片 Wafer 做 Compare 分析 (Post Adder, Missing, Common, Different, Sum Up).
- **叠图分析**: 支持叠图分析; 支持 Defect Map Overlay CP Map.
- **Kill Yield**: 支持 Defect Kill Yield Ratio 分析与报告.
- **测量工具**: 支持 Defect 量距.
- **Zone 分析**: 支持 Zone Map 分析, 包含 Zone 的定义、Zone 分析以及分类.
- **Small Die**: 支持 Small Die 功能.
- **WIP 关联**: 支持 By Lot/Wafer Cross WIP 分析.

#### 1.4 Defect Image

支持 Defect 图像的树形浏览、排序、查看与批量操作, 并可与 Map/Die 交互联动.

- **树形浏览**: 支持树形结构快速查看 Image.
- **排序功能**: 支持 Image 升降排序 (By Scan Time, By Layer, By Defect ID, By Wafer ID).
- **文本显示**: 支持 Image 自定义文本显示功能, 支持一键展示/隐藏文本.
- **图像操作**: 支持双击放大 Image 及切换 Image; 支持 Image 放大、缩小、旋转; 支持 Image Copy 功能.
- **选择联动**: 支持单选、多选、连选 Defect Image, 并与 Map, Die 交互.
- **重新分类**: 支持单选、多选 Defect 进行 Reclass.

#### 1.5 Defect Chart & Wafer Summary

支持缺陷统计图表绘制与汇总表格分析, 提供灵活的运算、排序与显示配置.

- **图表类型**: 支持 Trend, BarStack, BoxPlot 图表分析; 支持 Split 分析.
- **层级绘制**: 支持 Lot, Wafer 以及 Layer 级别的 Chart 绘制.
- **运算公式**: 支持多种运算公式 (Sum, Average, Max, Min, Median, Std).
- **表格操作**: 表格数据支持 Copy; 表格列支持自定义展示/隐藏; 表格支持删除行; 表格、Chart 数据支持升降序排列.

#### 1.6 Klarf File Viewer

支持 Klarf 文件导入分析.

### 2. 其他功能

#### 2.1 基本要求

支持多类型数据的收集、处理、可视化与缺陷分析, 并能自动生成分析报告.

- **数据处理**: 系统能提供数据处理功能, 能收集处理 History/WIP, WAT, CP, Metrology (EDC), Event, FT, Defect, Defect Image 等类型的数据.
- **数据可视化**: 系统能提供数据可视化功能, 如数据表格、图表、Wafer Map.
- **缺陷分析**: 系统能提供 Defect 数据的识别、分类、运算以及分析功能.
- **报告生成**: 系统能提供报告生成功能, 能就分析过程中产生的结果自动生成 Excel, PPT 格式的报告.

#### 2.2 数据加载程序

支持从文件、业务系统及数据库加载数据, 提供高效可靠的 Loader 开发平台与 Defect Classification 维护能力.

- **多源加载**: 数据加载程序支持从文件中加载数据 (WAT, CP, FT, Defect 等类型); 支持从业务系统 (如 MES) 加载数据; 支持从其它数据源 (如 DB) 加载数据.
- **Loader 平台**: 提供高效、可靠的数据加载程序开发平台 (Loader).
- **日志服务**: 数据加载程序提供详细日志服务.
- **自动 DSA**: 加载程序自动计算 DSA 功能.
- **Classification 维护**: 支持 Defect Classification 的维护, 包括增加、删除、更改; 支持从 KLARF 中导入 Defect Classification 定义.
- **ADC 整合**: 支持整合 ADC Defect 分类结果数据.

## WJS Function List

### 1. 系统管理

#### 1.1 权限与组织管理

支持用户、角色、菜单及组织架构的维护, 实现系统的权限管控与基础组织配置.

- **角色管理**: 支持为用户区分角色类型, 配置菜单功能权限, 增删改查角色.
- **用户管理**: 支持用户的增删改查.
- **菜单管理**: 支持 WJS 左侧菜单栏的增删改查.
- **部门管理**: 支持部门的增删改查.
- **工厂管理**: 支持工厂或车间的增删改查.

#### 1.2 基础数据设置

支持邮件组、栏位、类表、类表值及区域等基础数据的维护, 为策略与 Job 分类提供支撑.

- **邮件组设置**: 支持邮件组的增删改查.
- **栏位设置**: 支持栏位的增删改查, 与类表设置、类表值设置结合使用, 为基础数据.
- **类表设置**: 支持类表的增删改查, 与栏位设置、类表值设置结合使用, 为基础数据.
- **类表值设置**: 支持类表值的增删改查, 与类表设置、栏位设置结合使用, 为基础数据.
- **区域设置**: 支持区域的增删改查, 用于对 Job 进行分类.

#### 1.3 任务管理

支持管制限的定时自动重算, 保持管控参数的时效性.

- **管制限重算**: 定时对管制计划设定 (Job) 的控制限进行重新计算, 根据现有 Job 里的数据进行管控限重算, 支持每隔一段时间计算一次或按数据点数重新计算两种重算模式.

### 2. 产品信息管理

#### 2.1 产品

支持产品信息的维护与同步.

- **产品维护**: 支持产品的增删改查.
- **MES 同步**: 支持同步 MES 产品信息.

#### 2.2 产品参数设置

支持产品参数的维护, 用于策略判定与图表上下限显示.

- **参数维护**: 支持参数的增删改查, 用于策略判定和 By Lot Chart 上下限显示.
- **批量导入导出**: 支持产品参数的批量导入导出.

#### 2.3 重新计算结果设定

支持将重算控制限应用到 Job, 并通过浮动操作灵活调整管控限.

- **结果应用**: 用户可将计算出的控制限结果应用到 Job 的控制限, 以便收紧管控限.
- **浮动操作**: 支持手动浮动操作, 按比例手动扩大或收紧管控限; 支持多条重算结果同时浮动操作.
- **历史查询**: 支持浮动历史查询.

### 3. 数据监控

#### 3.1 CP 策略

支持 CP 卡控规则的设定, 对 CP 原始数据进行处理, 并可与 MES 联动执行管制动作.

- **CP Judge**: 支持 CP Judge 的增删改查和复制功能, 用来设定 CP 的卡控规则, 根据卡控规则对 CP 原始数据进行处理.
- **邮件警报**: 支持多个邮件组设定, 产生 Alarm 时合并邮件组成员依次发送警报邮件.
- **Action 联动**: 支持 Rule 逐条设定 Action 与 MES 联动, 实现 Hold Lot 或 OCAP 等.
- **Rule 去重**: 支持 RuleId 多条重复设定, 根据具体设定值去重.

#### 3.2 SPC 管制设定

支持 SPC Chart 的新增与配置, 涵盖参数设定、控制图与管控规则三大模块, 并可与 MES 联动.

- **Chart 新增**: 支持新增 Chart, 包括参数设定、控制图、管控规则三大模块.
- **控制图**: 支持设定规格线、控制线; 控制图包括计量型控制图 (Xbar, R, S, X, MR, 平均值、标准差、中位数、合格率、失效率等).
- **管控规则**: 支持设定 Chart 的管控规则, 并支持设定每条管控规则对应的管制动作 (Hold Lot, Hold Equipment, OCAP), 与 MES 进行 Action 联动.
- **维护功能**: 支持增删改查和复制, 支持批量导入导出及模板下载, 可查看 Job 的历史修改信息.
- **多邮件组**: 支持多邮件组设定.
- **规则配置**: 支持判异规则按需选用, 灵活设置, 可以增加或减少判异规则或管制动作 (基于对工艺的研究程度).

#### 3.3 PCM 策略

支持 PCM 卡控规则的设定, 对 PCM 原始数据进行处理, 并可与 MES 联动执行管制动作.

- **PCM 策略**: 支持 PCM 策略的增删改查和复制功能, 用来设定 PCM 的卡控规则, 根据卡控规则对 PCM 原始数据进行处理.
- **邮件警报**: 支持多个邮件组设定, 产生 Alarm 时合并邮件组成员依次发送警报邮件.
- **Action 联动**: 支持 Rule 逐条设定 Action 与 MES 联动, 实现 Hold Lot 或 OCAP 等.
- **Rule 去重**: 支持 RuleId 多条重复设定, 根据具体设定值去重.

#### 3.4 SPC Chart

支持单 Job 与多 Job 控制图的查询、交互操作与显示自定义, 实现过程监控与异常处理.

- **Chart 查询**: 支持根据查询条件查询单 Job 的控制图或多 Job 的控制图; 支持根据工厂、区域、时间段、显示点数、查询剔除点、查询失效 Job 等多维度查询控制图.
- **点位筛选**: 支持根据过滤条件筛选控制图上符合条件的点, 将筛选后的点重新显示在控制图上.
- **Sample 点菜单**: 支持控制图上右键点击 Sample 点展示功能操作菜单, 进行相应功能操作; 可查询控制图对应采集批次的基本信息, 包括原始数据、客户、MFG、批号、当前控制限、当前判异规则、判异批次及整改措施、动态 Ca, Cp 及 Cpk 等.
- **导出功能**: 控制图支持导出图片、数据点及样点信息.
- **缩放操作**: 在控制图上按住鼠标左键向右下方拖动可实现控制图的放大, 反方向拖动则恢复原图大小.
- **数据点操作**: 支持控制图上数据点的备注、启用、排除、隐藏; 支持排除部分异常数据后重新计算.
- **显示自定义**: 支持控制图管控限颜色样式的自定义配置; 支持统计量显示栏的设定; 支持设置 XY 轴的范围和图形的转换, 包括在控制图旁边显示直方图和盒须图.

#### 3.5 By Lot Chart

支持 PCM, CP 和 FT 数据 By Lot 显示在 Chart 上, 图形类型包括 XBAR, Sigma, 直方图和盒须图, 并计算统计值 (Cp, Cpk 等).

#### 3.6 RuleSetting

支持 CP 策略与 PCM 策略用到的规则定义, 包括规则描述和规则 ID 等, 具体设定值在策略里体现.

### 4. 报表

#### 4.1 PCM 报表

支持 PCM 全参数报表, 根据批次号和时间段查询 PCM Judge 逻辑判断之后产生的全参数报表数据, 支持报表导出功能.

#### 4.2 PCM 客户参数报表

支持根据批次号和时间段查询 PCM Judge 逻辑判断之后产生的客户参数报表数据, 支持报表导出功能.

#### 4.3 PCM 关键参数报表

支持根据批次号和时间段查询 PCM Judge 逻辑判断之后产生的关键参数报表数据, 支持报表导出功能.

#### 4.4 PCMRawData 报表

支持查询 PCM 参数及点位原始数据.

#### 4.5 CP 报表

支持 CP 策略结果表, 包括良率、各参数失效率、各参数平均值、中位数和标准差及违反的 Rule.

#### 4.6 CPRawData 报表

支持查询 CP 各参数及点位原始数据.

#### 4.7 FT 报表

支持 FT 报表, 包括良率、各参数平均值、中位数和标准差.

#### 4.8 统计报表

支持 PCM 策略逻辑判断之后产生的报废与通过的片号详细报表, 支持报表导出功能.

### 5. WJS Loader

#### 5.1 WJS 获得 Lot 信息

支持从 MES 获取 CP TEST 和 WAT TEST 工步批次的详细信息, 根据 Lot 详细信息找到需要解析的所有片文件或批文件.

#### 5.2 WJS Call MES 接口 (WSDL)

支持调用 MES 接口将策略结果及 Action 标志位信息给到 MES, 实现批次自动过站、Lot Hold 或 Lot OCAP.
